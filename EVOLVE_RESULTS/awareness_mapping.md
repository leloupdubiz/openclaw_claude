## MAPPING 19 SUB-AVATARS

### CORE A1 — Die Erschöpfte Berufstätige

#### A1-a — Die Manager mit Reizbarkeit (La Manager Irritable)
**Awareness Level dominant : PROBLEM AWARE → SOLUTION AWARE**
- Elle sait qu'elle est stressée et que "quelque chose a changé"
- Elle fait le lien avec son travail mais pas encore avec le cortisol
- Elle a peut-être cherché "Stress abbauen", "Reizbarkeit was tun"
**Type d'ad recommandé :** Valider son problème (irritabilité = cortisol, pas le caractère) → proposer la solution biologique
**Hook PROBLEM AWARE :** *"Wenn dich Emails aufregen die früher harmlos waren — das ist kein Charakterfehler."*
**Hook SOLUTION AWARE :** *"Cortisol macht gereizt. KSM-66 senkt es nachweislich."*

---

#### A1-b — Die Freelancerin ohne Ausschaltknopf
**Awareness Level dominant : PROBLEM AWARE**
- Elle sait que son cerveau "ne s'arrête pas" mais ne sait pas pourquoi
- N'a pas encore le terme "cortisol" dans son vocabulaire
**Type d'ad recommandé :** Nommer le problème + expliquer le mécanisme
**Hook PROBLEM AWARE :** *"Dein Laptop ist zu. Dein Kopf läuft weiter. Das hat einen biologischen Grund."*

---

#### A1-c — Die Ärztin / Pflegerin
**Awareness Level dominant : SOLUTION AWARE → PRODUCT AWARE**
- Professionnelle de santé → comprend le mécanisme cortisol
- Cherche une solution naturelle validée scientifiquement
**Type d'ad recommandé :** Mécanisme scientifique direct + étude PMC
**Hook PRODUCT AWARE :** *"Als Medizinerin weißt du was Cortisol macht. KSM-66 ist die klinische Antwort."*

---

#### A1-d — Die Karrierefrau nach 40
**Awareness Level dominant : PROBLEM AWARE → SOLUTION AWARE**
- Sait qu'elle a "changé depuis 40 ans" mais attribue à la ménopause seule
- Pas consciente du rôle dual cortisol + œstrogènes
**Type d'ad recommandé :** Révéler le mécanisme caché (cortisol + hormones)
**Hook PROBLEM AWARE :** *"Mit 40 wachst du nachts auf. Dein Arzt sagt Wechseljahre. Die Hälfte ist Cortisol."*

### CORE A2 — Die Überlastete Mutter

#### A2-a — Die Alleinerziehende
**Awareness Level dominant : UNAWARE → PROBLEM AWARE**

- Pense que c'est "normal d'être fatiguée" avec des enfants
**Type d'ad recommandé :** Symptôme pur → créer l'éveil (pas de mention produit pendant 5s+)
**Hook UNAWARE :** *"Du machst das alleine. Dein Körper arbeitet rund um die Uhr. Auch nachts."*

**⚠️ ATTENTION :** Exclure les femmes allaitantes — contre-indication BfR

#### A2-c — Die Sandwichgeneration-Mutter

## RÈGLE DE CRÉATION D'AD PAR AWARENESS LEVEL

### UNAWARE — Script type
```
Hook : [Symptôme que l'avatar se dit lui-même] (pas de mention produit)
Corps : Expliquer que ce symptôme a UNE CAUSE précise (le cortisol)
Solution : Nellio arrive comme la réponse logique au mécanisme
CTA : Doux — "Erfahre mehr" ou "Teste risikofrei"
```

### PROBLEM AWARE — Script type
```
Hook : Valider le problème (il sait qu'il a un problème, on valide)
Corps : Nommer la cause (cortisol) + expliquer POURQUOI ses tentatives ont échoué
Solution : Nellio avec le mécanisme KSM-66 comme résolution
CTA : Moyen — "45 Tage oder Geld zurück"
```

### SOLUTION AWARE — Script type
```
Hook : Différencier (pas "encore un supplement" mais "la différence est dans le mécanisme")
Corps : KSM-66 vs generic / liquid vs capsule / synergie 4 ingrédients
Solution : Preuve (étude, 20K reviews, 4.8/5)
CTA : Fort — "Jetzt risikofrei testen"
```

### PRODUCT AWARE — Script type
```
Hook : Comparaison directe (KSM-66 spécifique, pas générique)
Corps : Données cliniques + différenciateur unique Nellio
Solution : Offre + garantie
CTA : Direct — "Kaufe jetzt — 45 Tage Geld-zurück-Garantie"
```
