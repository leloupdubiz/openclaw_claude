## 4. SPECS TECHNIQUES TIKTOK ADS

### Formats
| Format | Dimensions | Durée | Usage |
|--------|------------|-------|-------|
| In-Feed Ad | 1080×1920 (9:16) | 5-60s | Principal |
| TopView | 1080×1920 | 5-60s | Brand awareness (cher) |
| Spark Ad | Native | Variable | Booster un post organique |

### Règles techniques
- Résolution : 1080×1920 minimum
- Pas de bandes noires (fill cadre complet)
- Sous-titres intégrés : taille minimale 40px, contraste fort
- CTA text overlay : minimum 3 dernières secondes
- Musique : utiliser bibliothèque commerciale TikTok (éviter droits)
- Son actif : obligatoire (contrairement à Meta)

### Budget test recommandé
