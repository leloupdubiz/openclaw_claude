## 🚨 PROBLÈMES FRÉQUENTS & SOLUTIONS

### Problème : Refus Meta "health claim"
**Solution :** Retirer "Schlafstörungen" du visuel · Remplacer par "Schlafqualität" ou "Erholung" · Le body copy peut garder "Schlaf" si pas de claim thérapeutique

### Problème : Budget non dépensé (ad set stuck)
**Solution :** Vérifier audience size (doit être >500K pour Broad) · Tenter un nouveau ad set avec "Interest : Schlaf" uniquement

### Problème : Pixel ne fire pas
**Solution :** Ouvrir drinknellio.com → F12 → Network → filtrer "pixel" → tester un ajout panier · Si Pixel Helper Extension montre rouge → réinstaller le pixel Shopify

### Problème : CPM > €20 (très haut)
**Solution :** Limiter les heures de diffusion (8h-22h seulement) · Exclure audience ayant déjà visité le site (retarget séparé) · Élargir l'audience WSD-2

### Problème : OYONO occupe les positions top
**Solution :** Primary text Version B activé ("20.000 Bewertungen vs 1 Expertin") · Budget WSD-3 (Women 28-50) augmenté → audience OYONO moins présente sur ce segment
