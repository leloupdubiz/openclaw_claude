## STEP 2 — PRODUCT IN ACTION

### Version Expert (pour educated audience / long-form ads)

Nellio UltraCalm régule le cortisol via **4 mécanismes simultanés** :

**Mécanisme 1 — L-Theanin (action rapide : 60 min)**
- Inhibe L-glutamate (neurotransmetteur excitateur) via récepteurs NMDA/AMPA
- Augmente GABA endogène (neurotransmetteur inhibiteur)
- Active ondes alpha EEG (relaxation sans somnolence mesurée à EEG)
- À 400mg : active également ondes Gamma → amélioration cognitive
- Bloque les pics AIGUS de cortisol dès la première prise

**Mécanisme 2 — KSM-66® Ashwagandha (action profonde : 2-8 semaines)**
- Module l'axe HPA (Hypothalamo-Hypophyso-Surrénalien)
- Réduit sécrétion de CRH → moins d'ACTH → moins de cortisol surrénalien
- Résultat mesuré : **-27.9% cortisol sérique** (PMC 31517876, 60 jours)
- Inhibe 11β-HSD1 dans tissu adipeux → réduit graisse viscérale
- Augmente BDNF → régénération neuronale (neuroplasticité)
- Normalise CAR (Cortisol Awakening Response) → énergie matinale restaurée

**Mécanisme 3 — Magnésium Glycinate (réparation : 1-3 semaines)**
- Rompt le cercle vicieux stress → épuisement Mg → plus de stress
- Active récepteurs GABA-B via glycine (double action calmante)
- Réduit cortisol nocturne → améliore sommeil profond
- 90% biodisponibilité vs 4% pour MgO (forme économique concurrents)
- Réduit IL-6 (marqueur inflammation liée au stress chronique)

**Mécanisme 4 — Vitamine D3 (amplificateur : dès J7)**
- Active VDR (récepteurs D3) dans l'hypothalamus → module axe HPA
- Amplifie efficacité KSM-66® de +30-40%
- Régule synthèse de sérotonine → humeur stable
- 75% Deutschen déficients → correction silencieuse mais massive
