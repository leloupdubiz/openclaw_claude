## ⚡ SETUP CANVA (1 fois)

1. **Ouvrir Canva.com** → "Créer un design" → "Taille personnalisée"
2. **Format Feed :** 1080 × 1080 px (ou template "Publication Instagram")
3. **Format Story :** 1080 × 1920 px (ou template "Story Instagram")
4. **Dupliquer** le template Feed pour les 3 visuels → adapter par visuel

---

## 🎨 CHARTE GRAPHIQUE NELLIO (à respecter sur les 3 ads)

| Élément | Valeur |
|---------|--------|
| Fond principal | Bleu nuit `#0B1628` |
| Texte principal | Blanc `#FFFFFF` |
| Texte secondaire | Gris clair `#CBD5E1` |
| Accent / Badge | Jaune chaud `#F5C842` |
| Font principale | Montserrat Bold (disponible Canva) |
| Font secondaire | Montserrat Regular |
| Logo Nellio | Blanc, bas gauche, taille ~8% de la largeur |

**Produit photo :** Utiliser le shot produit officiel (pot UltraCalm sur fond blanc ou neutre). Si pas disponible → mockup Canva avec texte produit.

## 📤 EXPORT & UPLOAD META

### Export depuis Canva
1. "Télécharger" → PNG (qualité max) → ou JPG si < 1MB
2. Nommer selon convention : `NLO_DE_STA01_Feed_1080.png`
3. Vérifier : minimum **1080×1080 px**, < **30MB**

### Upload Meta Ads Manager
1. Ads Manager → Créatifs → "Ajouter un média"
2. Pour le Carousel : créer une campagne Carousel séparée (3 slides = 3 "cartes")
3. Associer les Headlines depuis ce fichier (copier-coller)
4. UTM à ajouter sur les URLs : `?utm_source=meta&utm_medium=paid&utm_campaign=sta02&utm_content=STA01_BC-A_H1`
