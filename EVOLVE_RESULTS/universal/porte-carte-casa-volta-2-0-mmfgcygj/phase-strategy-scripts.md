## STRUCTURE SCRIPT EVOLVE (6 étapes obligatoires)

```
1. HOOK        (0-3s)   → Interrompre le scroll
2. PROBLEM     (3-8s)   → Amplifier la douleur connue
3. BRIDGE      (8-12s)  → Transition vers la solution
4. MECHANISM   (12-20s) → Montrer comment ça marche (le DIFFÉRENCIANT)
5. PROOF       (20-25s) → Preuve sociale / résultat concret
6. CTA         (25-30s) → 1 action claire, 0 friction
```

## SCRIPT S1 — "LE TEST HONNÊTE" (30 secondes)
*Angle A1 · Avatar : Homme 28-42, Solution-Aware · Format : UGC face-cam*

```
[0-3s]   HOOK VISUEL : sortir un vieux portefeuille épais de sa poche arrière + le poser lourdement
         HOOK TEXTE (sous-titre) : "J'ai testé 5 wallets slim. Voilà ce que personne ne dit."

[3-8s]   PROBLEM : "Franchement, la plupart c'est du marketing.
                    Le Secrid grince après 2 ans.
                    Le Ridge fait rentrer les cartes à la force.
                    Et les autres... tu vois."

[8-12s]  BRIDGE : "Jusqu'à ce que j'essaie le CasaVolta.
                   Là j'ai compris pourquoi il avait 4,8 sur 5 avec 362 avis."

[12-20s] MECHANISM : [POV] Bouton latéral → pression → 7 cartes éjectées en cascade
                     "Appuie. Une fois. Tes 7 cartes sortent en éventail.
                      Tu prends celle que tu veux. Tu relâches. C'est tout."

[20-25s] PROOF : "362 personnes l'ont noté 4,8. Pas 400+. 362, vérifiés.
                  Mon ancienne marque ? 3,4. Cherchez."

[25-30s] CTA : "Lien en bio. 60 jours pour changer d'avis. Ou remboursé."
```

**Notes réalisation :**
- Filmer en lumière naturelle fenêtre, pas en studio → crédibilité UGC
- Le créateur parle directement caméra, pas de script apparent
- Montrer le vieux portefeuille en contraste (idéalement un bi-fold usé)
- Slow-mo sur la cascade de cartes (0.3x vitesse)
- Sous-titres blancs, pas de template YouTube
