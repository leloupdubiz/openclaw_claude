## RÈGLE DES 3 SECONDES (EVOLVE)

Un hook = **la première image ou phrase** qui crée un GAP DE CURIOSITÉ suffisant pour que le viewer arrête de scroller.

```
HOOK GAGNANT = Interruption + Promesse implicite + Tension non résolue
HOOK RATÉ    = Description du produit + Prix + CTA
```

**Types de hooks visuels (vidéo) :**
- Ideal outcome : montrer la vie AVEC le produit (pas le produit)
- Weird clip : quelque chose d'inattendu (mécanisme en slow-mo)
- Problem demo : montrer la douleur sans nommer le produit
- Product demo : montrer le mécanisme pur (pour Product-Aware)
