## 4. CLAIMS DÉFENDABLES vs À ÉVITER

### **Contexte Réglementaire FR/DE** :
- 🇫🇷 **France** : DGCCRF strict ; Health claims (Art. L4113 du Code de la Santé Publique) = "détox" ❌, "nettoyer le foie" ⚠️ (borderline), "soutenir la fonction hépatique" ✅
- 🇩🇪 **Allemagne** : AMG + AVV restrictions ; "Leber-Detox" technically OK if substantiated; "heilt Leberkrankheiten" ❌; "Unterstützt die Lebergesundheit" ✅
- **EU-wide** (EFSA, Regulation 1169/2011) : Nutrient claims require substantiation ; Disease claims = pharmaceutical (forbidden)
