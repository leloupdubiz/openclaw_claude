# Meta Comments Moderation Guide DE — NLO_DE
> Clawdbot Prime ⚡ | Généré : 2026-03-04 | Usage : avant lancement Meta

---

## CONTEXTE

**Pourquoi ce document :** Dès que les premières ads Meta sont publiées, les commentaires arrivent dans les 24h. Sans stratégie, 3 risques :
1. Commentaires négatifs visibles → ROAS -15 à -25% (preuve sociale négative)
2. Concurrents qui postent dans les comments → audience détournée
3. Meta qui flag le contenu si trop de signalements → ad en review

Ce guide = réponses prêtes à copier-coller + règles de modération + stratégie pour maximiser la preuve sociale des comments.

---

## §1 — RÈGLES DE MODÉRATION (PRIORITÉS)

### Règle d'or
> Un commentaire négatif sans réponse dans les 4h = 3× plus d'impact négatif qu'avec réponse.

### Matrice de décision

| Type de commentaire | Action | Délai |
|---------------------|--------|-------|
| Scepticisme normal ("funktioniert das wirklich?") | Répondre + laisser | < 2h |
| Témoignage positif | Répondre avec emoji + laisser | < 1h |
| Négatif récupérable ("wirkt nicht") | Répondre privé + laisser | < 2h |
| Attaque produit / faux avis | Répondre public + masquer | < 1h |
| Concurrent visible | Masquer immédiatement | ASAP |
| Spam / bot | Supprimer | ASAP |
| Question légitime | Répondre public | < 3h |
| Commentaire raciste / haineux | Supprimer + signaler | ASAP |
| Demande remboursement public | DM vers SAV + masquer | < 1h |

## §3 — STRATÉGIE PREUVE SOCIALE COMMENTAIRES

### Objectif : transformer les comments en actif publicitaire

**Phase 1 (J0-J7) — Seeding social proof :**
- Demander à 5-10 proches de commenter "authentiquement" sur les ads (expériences vraies seulement)
- Répondre à TOUS les commentaires dans les 24h — signal Meta algo positif
- Épingler (si possible) un super-commentaire positif en haut de l'ad

**Phase 2 (J7+) — Amplification organique :**
- Screenshotter les meilleurs commentaires → intégrer dans prochains ads (body copy ou overlay)
- Format "Ce que nos clients disent :" → 3 citations extraites des commentaires
- Les meilleurs verbatims commentaires → alimentent VERBATIMS database (avec permission)

## §4 — GESTION DE CRISE COMMENTAIRES (URGENCES)

### Scénario A : Commentaire viral négatif (100+ likes en 2h)
1. Masquer immédiatement
2. Répondre en DM à l'auteur (bonne foi obligatoire)
3. Si faux → signaler à Meta via le bouton "Report" sur le commentaire
4. Si vrai → rembourser + proposer de modifier son commentaire

### Scénario B : Campagne coordonnée de signalements (bot/concurrent)
1. Ne pas paniquer — Meta est habitué, les flagging abusifs sont détectés
2. Contacter Meta Business Support via business.facebook.com
3. Documenter les commentaires suspects (screenshots)
4. Mettre l'ad en pause 24h si reviews → redémarrer après

### Scénario C : Question médicale sérieuse (interaction médicaments, santé)
1. Répondre publiquement (SAV script T5)
2. NE PAS donner de conseil médical
3. Toujours rediriger vers médecin pour les questions sérieuses
4. Légal DE : §7 HWG (Heilmittelwerbegesetz) interdit les affirmations thérapeutiques

## §5 — SETUP TECHNIQUE META

### Paramètres recommandés avant lancement

**Facebook Page Settings :**
- Filtres de mots (**Meta Page Settings > Profanity Filter**) → activer "Strong"
- Liste de mots bloqués : [concurrents noms, termes offensants, "Betrug", "Fake", "Scam"]
- Auto-masquage : activer pour questions personnelles santé

**Meta Ads Manager :**
- Commentaires Meta ads : **Page settings > Branded Content** → contrôle des commentaires
- Modération centralisée via **Meta Business Suite > Inbox** (regroupe tous les commentaires)

**Outil recommandé (optionnel) :**
- **Agorapulse** ou **Later** → notifications instantanées comments → réponse mobile en 5 min
