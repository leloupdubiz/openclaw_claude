Voici une résumé de la vidéo "How To Run Facebook Ads in 2024 Complete Guide" de Davie Fogarty, fondateur d'Oodie, avec des applications pour drinknellio.com :

**Étape 1 : Créer un compte Business Manager**

* Allez à business.facebook.com et créez un compte Business Manager
* Créez un nouveau compte publicitaire dans le Business Manager
* Nommez votre compte publicitaire avec le nom de votre marque et la région

**Étape 2 : Vérifier votre identité et ajouter des utilisateurs**

* Vérifiez votre identité et ajoutez des utilisateurs pour gérer le compte publicitaire
* Ajoutez votre ABN si vous êtes un résident australien

**Étape 3 : Créer un pixel**

* Allez à Shopify et créez un nouveau pixel
* Connectez votre pixel à votre compte publicitaire
* Nommez votre pixel avec le nom de votre marque

**Étape 4 : Configurer le pixel**

* Configurez le pixel pour suivre les événements de conversion (achat, ajout au panier, etc.)
* Configurez le pixel pour suivre les événements de retargeting (vues de produit, etc.)

**Étape 5 : Créer une campagne**

* Allez à l'Ads Manager et créez une nouvelle campagne
* Nommez votre campagne avec un nom clair et concis
* Sélectionnez l'objectif de la campagne (conversion, etc.)

**Étape 6 : Configurer l'ad set**

* Configurez l'ad set pour cibler l'audience appropriée
* Sélectionnez les intérêts, l'âge et le sexe de l'audience
* Configurez les placements automatiques ou manuels

**Étape 7 : Créer une publicité**

* Créez une nouvelle publicité dans l'ad set
* Sélectionnez le contenu de la publicité (image, texte, etc.)
* Configurez les paramètres de la publicité (budget, cible, etc.)

**Applications pour drinknellio.com**

* Créer un compte Business Manager pour gérer les publicités
* Configurer un pixel pour suivre les événements de conversion
* Créer une campagne pour cibler l'audience appropriée
* Configurer l'ad set pour cibler l'audience appropriée
* Créer une publicité pour promouvoir les produits de drinknellio.com

**Citation clé**

* "La clé pour réussir avec les publicités Facebook est de comprendre comment les utilisateurs interagissent avec votre marque et de cibler l'audience appropriée." - Davie Fogarty
