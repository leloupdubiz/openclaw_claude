# 📋 🆕 The Evolve Growth Guide Overview - 👣 Evolve · Evolve
> Module 05 — How To Plan Ads | EVOLVE MARKETING

**Concept Central**

- La formation EVOLVE Marketing propose une nouvelle approche pour planifier et créer des publicités efficaces.
- L'objectif est de simplifier le processus de création d'ads en mettant en avant les éléments clés.
- L'utilisation de l'entraînement Avatar pour créer des avatars et identifier les mécanismes de test est un élément clé de cette approche.

**Points Clés**

- La formation EVOLVE Marketing propose deux guides de croissance : l'un pour planifier les ads et l'autre pour les créer.
- Le guide de croissance 1.0 est toujours utilisé à l'agence, mais a été optimisé pour se concentrer sur les éléments clés.
- Le guide de croissance 2.0 est une version simplifiée qui met en avant les éléments clés de l'entraînement Avatar.
- Les éléments clés du guide de croissance 2.0 incluent :
 + Auteur
 + Concept d'ad
 + Désir
 + Intention
 + Niveau de conscience
 + Type d'ad
 + Format d'ad
 + Lien vers le bref
 + Lien vers les résultats et les apprentissages de l'ad

**Insights Actionnables**

1. Simplifier le processus de création d'ads en mettant en avant les éléments clés.
2. Utiliser l'entraînement Avatar pour créer des avatars et identifier les mécanismes de test.
3. Mettre en place un système de suivi pour évaluer l'efficacité des ads.

**Applications pour drinknellio.com**

- Utiliser le guide de croissance 2.0 pour planifier et créer des publicités efficaces pour Nellio UltraCalm.
