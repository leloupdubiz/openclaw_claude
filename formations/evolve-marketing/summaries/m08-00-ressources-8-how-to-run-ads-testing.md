**Concept Central**

- Tester plusieurs créatifs pour trouver le bon
- Utiliser la méthode 3:2:2 pour tester des créatifs
- Tester des landing pages avec la méthode 3:2:2:2

**Points Clés**

- Utiliser un document de media buying pour planifier les campagnes
- Tester 3 créatifs, 2 copies et 2 headlines pour chaque ad set
- Utiliser les colonnes de performance pour évaluer les campagnes
- Lancer des campagnes avec un budget minimum de 100$/jour
- Tester les créatifs pendant au moins 3 jours avant de les arrêter
- Arrêter les créatifs qui ne fonctionnent pas bien après 7 jours
- Utiliser la méthode 3:2:2:2 pour tester les landing pages
