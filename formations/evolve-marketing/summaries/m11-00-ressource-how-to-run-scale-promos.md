# 🚀 Ressource How to run & scale promos
> Module 11 — Run & Scale Promos | EVOLVE MARKETING

**Concept Central**

- La stratégie de promotion de Nellio UltraCalm est axée sur les périodes de vente, comme la fête des Mères et les soldes de fin d'année.
- L'utilisation de campagnes CBO (Campaign Budget Optimization) pour optimiser les dépenses publicitaires.
- L'importance de surveiller les KPIs (indicateurs clés de performance) pour ajuster les dépenses publicitaires.

**Points Clés**

- Créer une campagne CBO avec 3 ad sets :
 + Un ad set large (Broad ad set)
 + Un ad set WARM60 (FB Engagers, IG Engagers, 95% Video Viewers sur les 60 derniers jours)
 + Un ad set HOT90 (Website visitors, view content, add to cart, initiate checkout sur les 90 derniers jours)
- Utiliser des bannières de vente pour les annonces existantes et ajouter des annonces simples avec des images de produits qui mentionnent la vente.
- Ne pas faire de retargeting sauf pendant les périodes de vente.
- Utiliser la méthode de surf scaling pour ajuster les dépenses publicitaires en fonction des KPIs.

**Insights Actionnables**

1. **Surveiller les KPIs** : comprendre les indicateurs clés de performance pour ajuster les dépenses publicitaires.

3. **Créer des ad sets ciblés** : créer des ad sets ciblés pour atteindre les clients potentiels qui sont susceptibles d'acheter.
