# 🏃 How To Create Rules For Promo Campaigns - 👣 Evolve · Evolve
> Module 08 — How To Run Ads | EVOLVE MARKETING

**🎯 Concept Central**

- Créer des règles pour gérer les campagnes publicitaires sur Meta Ads.
- Utiliser des règles pour désactiver les campagnes promotionnelles après une date spécifique.
- Gérer les budgets et les paramètres de publication pour les campagnes publicitaires.

**📌 Points Clés**

- Créer une nouvelle règle en cliquant sur "Créer une nouvelle règle" dans la section "Règles" de l'espace publicitaire.
- Sélectionner le type de règle (par exemple, "Règle personnalisée") et cliquer sur "Suivant".
- Définir le nom de la règle et le type de campagne à laquelle elle s'applique (par exemple, "Campagnes", "Sets d'annonces" ou "Annonces").
- Définir l'action à effectuer lorsque la règle est activée (par exemple, "Désactiver" ou "Augmenter le budget").
- Définir la condition pour laquelle la règle est activée (par exemple, "Date actuelle supérieure à une date spécifique").
- Configurer les paramètres de publication et les budgets pour la campagne.

**💡 Insights Actionnables**

1. Utiliser les règles pour gérer les campagnes promotionnelles et éviter les erreurs de publication.
2. Configurer les règles pour désactiver les campagnes après une date spécifique pour éviter les dépenses inutiles.
3. Utiliser les règles pour ajuster les budgets et les paramètres de publication en fonction des résultats de la campagne.
