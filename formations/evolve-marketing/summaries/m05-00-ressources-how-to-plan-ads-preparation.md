**Concept Central**

* Définitions clés pour planifier des publicités efficaces
* Comprendre les concepts, les angles, les variations et les formats

**Points Clés**

* **Concept** : La grande idée derrière les publicités que vous souhaitez tester
* **Batch/Test #** : Le numéro de suivi pour chaque test
* **Angle** : La stratégie de vente - comment vous vendez le produit
* **Variations** : Les différentes façons de communiquer votre raison de vente
* **Format** : Le type d'annonce que vous créez

**Insights Actionnables**

1. Chaque annonce nécessite un angle. Même si votre concept est "tester des formats", vous devez définir ce que vous vendez.
2. Les variations sont les différentes façons de communiquer votre raison de vente.
3. Le format est le type d'annonce que vous créez.
