# 🚀 How To Create Rules For Promo Campaigns
> Module 11 — Run & Scale Promos | EVOLVE MARKETING

**🎯 Concept Central**
- Créer des règles pour les campagnes publicitaires sur Meta Ads.
- Utiliser les règles pour désactiver les campagnes promotionnelles après une date spécifique.
- Gérer les budgets et les paramètres de campagne avec des règles personnalisées.

**📌 Points Clés**
- Créer une nouvelle règle en cliquant sur "Créer une nouvelle règle" dans la section "Règles" de l'espace de travail Meta Ads.
- Sélectionner la campagne, les sets d'annonces ou les annonces individuelles auxquelles la règle s'appliquera.
- Définir l'action à effectuer (désactiver, activer, augmenter ou diminuer les budgets).
- Configurer la condition pour déclencher la règle (par exemple, date, coût par résultat, etc.).
- Enregistrer la règle pour qu'elle soit appliquée automatiquement.

**💡 Insights Actionnables**
1. Utilisez les règles pour automatiser la gestion de vos campagnes publicitaires et économiser du temps.
2. Définissez des règles pour désactiver les campagnes promotionnelles après une date spécifique pour éviter les dépenses inutiles.
3. Utilisez les règles pour ajuster les budgets de vos campagnes en fonction de vos objectifs et de vos performances.

**🏪 Applications pour drinknellio.com**
- Créer une règle pour désactiver la campagne promotionnelle de Nellio UltraCalm après la date de fin de la promotion.
- Utiliser les règles pour ajuster les budgets de la campagne en fonction des performances et des objectifs.
- Automatiser la gestion des campagnes publicitaires pour éviter les erreurs humaines et économiser du temps.
