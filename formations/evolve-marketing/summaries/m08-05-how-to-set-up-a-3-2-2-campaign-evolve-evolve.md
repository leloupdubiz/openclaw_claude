## 🎯 Concept Central
- La campagne 3 2 2 est la méthode principale de media buying pour drinknellio.com.
- Elle est composée d'une campagne principale (CBO) qui teste des publicités à l'intérieur.
- Les publicités sont lancées manuellement au lieu d'utiliser des DCTs (Dynamic Creative Templates).

## 📌 Points Clés
- Créer une campagne principale CBO avec un minimum de 100 $.
- Configurer la campagne pour maximiser le nombre de conversions.
- Utiliser des publicités manuelles au lieu de DCTs.
- Configurer les paramètres de ciblage pour les âges, les genres et les localisations.
- Utiliser les options de ciblage original pour sélectionner des groupes d'âge spécifiques.
- Configurer la localisation, l'âge et le genre pour les publicités.
- Utiliser la fonction "voir les contrôles d'audience dans les paramètres de publicité" pour accéder aux options de ciblage original.
