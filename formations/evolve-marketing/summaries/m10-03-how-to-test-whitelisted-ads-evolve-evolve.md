# 📄 How To Test Whitelisted Ads - 👣 Evolve · Evolve
> Module 10 — How To Run Whitelisted Ads | EVOLVE MARKETING

Voici un résumé du transcript de la formation EVOLVE Marketing :

## 🎯 Concept Central
- Tester les publicités blanchies (whitelisted ads) pour améliorer la performance des campagnes publicitaires.
- Utiliser les publicités blanchies pour augmenter la visibilité et la conversion.

## 📌 Points Clés
- Tester les publicités blanchies dans la campagne principale pour les petites budgets (< 2K par jour).
- Utiliser un campagne de contenu brut (raw content campaign) pour les publicités blanchies.
- Créer un campagne de publicités blanchies séparée pour les budgets plus élevés (> 2K par jour).
- Utiliser le CBO (Campaign Budget Optimization) pour optimiser les dépenses publicitaires.
- Tester les publicités blanchies en les lançant sur une page native pour évaluer leur performance.

## 💡 Insights Actionnables
1. Tester les publicités blanchies dans la campagne principale pour les petites budgets.
