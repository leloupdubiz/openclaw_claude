> Module 06 — How To Make Ads | EVOLVE MARKETING

**🎯 Concept Central**

- La création d'ads efficaces nécessite une compréhension approfondie des fondamentaux du marketing.
- La stratégie créative doit être analysée et comprise pour créer des ads qui "bloquent".
- Les 5 étapes de l'awareness sont essentielles pour comprendre comment les gens interagissent avec les produits.

**📌 Points Clés**

- Les 5 étapes de l'awareness :
 - Unaware : les gens ne connaissent pas qu'ils ont un problème.
 - Problem-aware : les gens connaissent le problème, mais pas la solution.
 - Solution-aware : les gens connaissent la solution, mais pas que vous la proposez.
 - Product-aware : les gens connaissent les produits similaires, mais pas pourquoi le vôtre est le meilleur.
 - Most-aware : les gens connaissent tout, mais vous devez les convaincre d'acheter.
- Les ads doivent être créés en fonction de l'étape d'awareness du public cible.
- Les stories, secrets et infotainment sont utilisés pour les ads Unaware.
- Les pain points et les solutions sont utilisés pour les ads Problem-aware.
- Les preuves de la qualité du produit sont utilisées pour les ads Solution-aware.
- Les preuves de la qualité et les prix sont utilisés pour les ads Product-aware et Most-aware.

**💡 Insights Actionnables**

1. Analysez votre public cible pour comprendre leur étape d'awareness.
2. Créez des ads en fonction de l'étape d'awareness de votre public cible.
3. Utilisez les stories, secrets et infotainment pour les ads Unaware.
4. Utilisez les pain points et les solutions pour les ads Problem-aware.
5. Utilisez les preuves de la qualité du produit pour les ads Solution-aware.

**🏪 Applications pour drinknellio.com**

- Créer des ads Unaware pour présenter Nellio UltraCalm comme une solution pour les gens qui ne connaissent pas qu'ils ont un problème de stress.
- Créer des ads Problem-aware pour présenter les avantages de Nellio UltraCalm pour les gens qui connaissent le problème de stress, mais pas la solution.
- Créer des ads Solution-aware pour présenter Nellio UltraCalm comme la solution pour les gens qui connaissent la solution, mais pas que vous la proposez.
- Créer des ads Product-aware pour présenter les avantages de Nellio UltraCalm par rapport aux autres produits similaires.
- Créer des ads Most-aware pour présenter les avantages de Nellio UltraCalm et les prix compétitifs.

**⚡ Citation Clé**

> "Si vous ne comprenez pas les fondamentaux du marketing, vous ne pourrez jamais créer des ads qui 'bloquent'."
