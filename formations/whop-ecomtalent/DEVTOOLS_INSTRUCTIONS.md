3. **Onglet "Console"**
4. **Colle le contenu** du fichier `DEVTOOLS_SCRIPT.js`
5. Appuie sur **Entrée**
6. Attends ~30 secondes
7. **Copie le JSON** affiché (ou il est copié auto dans le presse-papier)
8. **Envoie le JSON** dans le channel Discord **#ressources**
