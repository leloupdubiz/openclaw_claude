# TITRE — Module CRO : Template Test avec IntelliGems
## 🎯 Concept Central
Le template test est un type de test A/B qui permet de comparer différentes versions d'un modèle de page web, tel que la page d'accueil, les pages de collection, les pages de produit ou les pages personnalisées. IntelliGems est un outil qui intègre parfaitement avec Shopify pour faciliter la création et la gestion de ces tests.

## 📌 Points Clés
* Les templates tests sont idéaux pour tester des modifications sur les pages web sans affecter la version live.
* IntelliGems permet de créer des templates tests en quelques clics, sans nécessité de compétences en codage.
* Il est possible de tester différents éléments, tels que les bannières, les vidéos, les textes, etc.
* Les tests peuvent être lancés sur des pages spécifiques ou sur des groupes de pages.

## 💡 Insights Actionnables
* Utiliser les templates tests pour améliorer l'expérience utilisateur et augmenter les conversions.
* Tester différents éléments de la page pour identifier les meilleurs performants.
* Utiliser IntelliGems pour faciliter la création et la gestion des tests.
* Analyser les données pour prendre des décisions éclairées sur les modifications à apporter à la page.
