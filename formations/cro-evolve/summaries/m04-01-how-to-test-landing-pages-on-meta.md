# TITRE — Module CRO : Test de Page d'Atterissage sur Meta

## 🎯 Concept Central
Le concept central de cette vidéo est de montrer comment configurer un test de page d'atterissage sur Meta (Facebook) pour évaluer les performances d'une nouvelle page d'atterissage.

## 📌 Points Clés
- Créer un nouveau test de page d'atterissage dans Meta
- Configurer un test ABO (Ad Set Budget Optimization) pour forcer le déploiement de la page d'atterissage
- Utiliser un budget minimum de 50 $ par jour pour obtenir des données rapidement
- Assurer que le suivi des conversions est correctement configuré
- Utiliser la fonction de duplication d'annonce pour créer une nouvelle annonce avec la même configuration que l'annonce originale, mais avec la nouvelle page d'atterissage

## 💡 Insights Actionnables
- Il est important de configurer correctement le suivi des conversions pour éviter les faux positifs ou les faux négatifs
- Utiliser la fonction de duplication d'annonce pour créer une nouvelle annonce avec la même configuration que l'annonce originale, mais avec la nouvelle page d'atterissage
- Assurer que le budget est suffisant pour obtenir des données rapidement
