# CRO Basics — 04 : Tools Of The Trade
> Formateur : Spencer | Formation : EVOLVE CRO | Durée estimée : ~8 min

---

## 🎯 Concept Central

Stack CRO en 3 outils. Priorité absolue : **PageDeck** (landing pages rapides). IntelliGems pour le A/B testing scientifique (brands $100K+/mois). Heatmap.com pour analyser le comportement utilisateur et le revenu par clic.

---

## 🛠️ Les 3 Outils

### 1. PageDeck — ⭐ L'OUTIL PRIORITAIRE
> *"Les landing pages les plus rapides pour l'e-commerce."*

- **Prix** : ~$129-149/mois (code **EZA15** = -15%)
- **Pourquoi** : Élimine les développeurs, les fichiers Figma, le QA interminable
- **Focus** : Offre, messaging, contenu — les vraies variables qui font convertir
- **A/B testing** : Intégré (ou en cours d'intégration selon la date)
- **ROI comparé** : Spencer payait $1K-5K/mois de développeurs pour faire la même chose
- **Vs Replo, Unbounce, Funnelish** : PageDeck recommandé — spécialisé e-com, plus rapide
- **À signer tout de suite** : Oui — la formation entière est basée dessus

### 2. IntelliGems — Pour brands $100K+/mois
> *"Le meilleur outil de split-testing Shopify, pour ceux qui ont le volume."*

- **Prix** : ~$99/mois (code **EZ810** = -10% → ~$89/mois)
- **Usage standard** : A/B testing pages, thèmes, templates — remplace Google Optimize (disparu)
- **Usage avancé** : Split-testing prix et shipping ($499/mois) — recommandé si +$100K/mois
- **Reporting** : Interface simple, claire, actionnable
- **Limitation** : Nécessite 2-3 mois et beaucoup de trafic pour data statistiquement fiable
- **Priorité** : Après PageDeck, pas avant

### 3. Heatmap.com — Complémentaire
> *"Le seul heatmap qui montre le revenu par clic, pas juste les clics."*

- **Prix** : ~$89/mois (pas de code promo pour l'instant)
- **Différence clé** : Revenu par zone cliquée vs heatmaps classiques (uniquement clics)
- **Usage** : Identifier quels éléments génèrent réellement des ventes sur les landing pages
- **Priorité** : Nice-to-have — pas obligatoire pour débuter
- **Créateur** : Outil de Dylan Ander (expert CRO mentionné dans les vidéos précédentes)

## 🏪 Application pour drinknellio.com

- **À faire maintenant** : Souscrire PageDeck ($129/mois, code EZA15) → c'est la base de toute la suite de la formation
- **Ne pas** installer IntelliGems avant d'avoir le volume suffisant (~300+ orders/jour)
- **Heatmap.com** : À considérer une fois les premières landing pages lancées pour identifier les friction points
- **Stack complet quand prêt** : PageDeck (landing pages) + Meta Ads (trafic) + Heatmap (analyse) → boucle d'optimisation continue
