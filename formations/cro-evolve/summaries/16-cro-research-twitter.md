# CRO Research — Twitter (Follow-Along)

**Concept Central**  
Twitter est une source gratuite et constante de CRO best practices validées par les experts du secteur. Observer comment les CRO leaders testent et itèrent offre une direction stratégique sans copier-coller.

---

## Points Clés

1. **Top 3 CRO experts sur Twitter** (suivis & analysés) :
   - **Audit** : Design épuré orienté conversion, avant/après visuels, data-driven
   - **Dylan** (heatmap.com) : 5 split tests prioritaires sans code, impact revenue élevé
   - **Sean & Taylor** : Breakdowns visuels simples, navigation/categorization, design fonctionnel
   - **Oliver** : 12+ ans CRO (max conversions, approche différente d'Audit)

2. **Concept clé : Design ≠ Conversion**  
   Design doit enhancer la conversion, pas juste "faire cool" — ce point résume 80% de la différence entre amateur et expert

3. **5 Split Tests Immédiats (Dylan)** :
   - Headline tests (ROI maximal)
   - Product tile rearrangement (heatmap prove impact)
   - Hero image testing (6% de l'écran = optimal)
   - Buy box (add to cart visibility + design)
   - Product price size (18px = optimal d'après Baymarch studies)

4. **Méthodologie CRO** : Visual case studies → données → insights → application custom (ne pas copier bêtement)

---

## Insights

- Les vraies sources CRO sont rares et identifiables (Audit + Dylan = références sûres)
- Les frameworks publiés gratuitement sur Twitter sont souvent testés avec de vrais budgets clients
- L'observation itérative de concurrents via heatmaps > brainstorm interne
- "Simple surface products get boring" = test guided browsing (shop by hair type, use case, etc.)
