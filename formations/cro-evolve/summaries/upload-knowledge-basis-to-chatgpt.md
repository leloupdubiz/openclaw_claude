# How To Upload Knowledge Basis To Chat GPT

## Concept Central
**Créer un Custom GPT entraîné sur les données de ton client** = transformer du feedback brut en une source d'intelligence interrogeable en langage naturel. C'est une IA specialized pour **une** marque.

## Points Clés
- **Custom GPT = ChatGPT Pro ($20/mois)** avec 3 capacités clés :
  - Upload de fichiers (PDF, CSV, texte)
  - Instructions système customisées
  - Partage avec l'équipe ou clients
- **Étapes de création** :
  1. ChatGPT Pro → My GPTs → Create New → Configure
  2. Name + Description : "This GPT knows everything about [Client] customers"
  3. Instructions : colle le prompt de summary (lien fourni dans la formation)
  4. Upload Files : les PDFs des réponses texte (bonus entries) + éventuellement website data
  5. Save et Share le lien
- **Files à uploader** :
  - PDFs avec les réponses texte libre uniquement (pas les multiples-choix)
  - Les PDFs doivent être nettoyés (en-têtes removés, réponses seules)
  - Optionnel : données website, reviews clients, competitive research
- **Prompt template fourni** : instructions pour GPT sur comment structurer les summaries
- **Variation avec vs sans certaines data** : GPT peut produire légèrement différentes summaries selon ce qu'il connaît — re-run 2-3x pour quality

## Insights
- Un Custom GPT = **5x plus powerful** qu'un generic ChatGPT pour une marque (context + data = precision)
- Uploader tous les files d'un coup = plus rapide que file-by-file, mais peut créer du noise
- Les instructions du prompt sont critiques — un mauvais prompt = summary inutile
- Ce Custom GPT devient un **asset partageable** : équipe interne + clients peuvent l'interroger directement

## Applications drinknellio.com
- **Créer un Nellio Custom GPT** :
  - Name : "Nellio UltraCalm Customer Insights GPT"
  - Description : "This GPT knows everything about Nellio customers, their stress, sleep issues, and why they bought."
  - Upload : les réponses texte des bonus entries (journeys, motivations, problèmes)
  - Instructions : adapter le prompt pour Nellio (ex: focus sur stress/sleep/concentration resolution)
- **Interrogations pratiques du GPT** :
  - "What are customers saying about our product vs competitors?"
  - "What problems did customers have BEFORE using Nellio?"
  - "Which words/phrases do customers use to describe stress relief?"
  - "What should our next product features be based on feedback?"
  - "Generate 3 ad angles based on customer journeys"
- **Share avec** : Chef, creative team, media buyer — tous ont accès à la même source de truth client
