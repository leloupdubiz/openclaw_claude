# Upload Knowledge Basis ChatGPT

## Concept Central
Créer un résumé personnalisé pour les données de feedback à l'aide de ChatGPT.

## Points Clés
- Créer un GPT personnalisé pour résumer les données de feedback
- Configurer le GPT pour analyser les données de feedback sous forme de texte
- Télécharger les données de feedback sous forme de PDF pour les importer dans ChatGPT

## Insights
- Utiliser un GPT personnalisé peut aider à optimiser le processus de résumé des données de feedback
- Les données de feedback écrites sont plus importantes que les questions à choix multiples pour obtenir des informations précieuses

## Applications pour Nellio UltraCalm (drinknellio.com)
- Créer un GPT personnalisé pour analyser les données de feedback des clients de Nellio UltraCalm
- Utiliser les insights obtenus pour améliorer l'expérience utilisateur et la conversion sur le site web de Nellio UltraCalm
