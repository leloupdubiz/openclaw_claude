# Résumé : Organiser Sheet Feedback Data — Partie 3 (How To Organize Sheet With Feedback Data - Part 3)

## Concept Central
Les surveys customisées (contrairement aux templates standards) nécessitent une adaptation méthodique du sheet : vérifier chaque question, adapter chaque réponse possible, et documenter les décisions pour les futurs analystes. Flexibilité + rigueur = data utilisable même pour cas non-standard.

## Points Clés
- **Surveys customisées = double travail** :
  - Ne pas utiliser le template brut directement
  - Lire la survey officielle (VialSweep edit)
  - Adapter colonne par colonne pour matcher les réponses réelles

- **Questions multiple-choice** :
  - Dupliquer structure d'une question MC existante
  - Copy/paste colonnes + update question title
  - Update réponses possibles (must match exactly = spelling, capitalisation)

- **Drag & drop pour réordonner** : si questions dans mauvais ordre vs template, déplacer colonnes pour aligner

- **Written answers** (questions libres) :
  - Garder dans le sheet (pour summaries AI)
  - Marquer en noir vs vert pour MC (différentiation visuelle)

- **Customisation = exception** : après adaptation, ce template peut être réutilisé pour ce client à l'avenir

## Insights
- Customers avec 50+ produits = surveys hyper-spécifiques (pas generic)
- Même les small variations (typo) en réponses = zéro match dans le filtering
- Documentation = crucial (futurs team members peuvent débugging easier)

## Applications Drinknellio.com
1. **Si survey Nellio custom** :
   - Adapter pour categories produit propres à Nellio (UltraCalm vs autres formules)
   - Ajouter questions uniques (ex: "Qui t'a recommandé ?" vs generic "Source")
   - Map chaque question → réponses officielles de VialSweep

2. **Réutilisabilité** : une fois customisée pour Nellio = template prêt pour prochaines campagnes giveaway
