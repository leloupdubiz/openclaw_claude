# CRO Basics — 02 : Types Of CRO
> Formateur : Spencer | Formation : EVOLVE CRO | Durée estimée : ~18 min

---

## 🎯 Concept Central

Il existe **3 types de CRO**. Spencer recommande quasi-exclusivement le 3ème — l'**Ad-Focused CRO** — pour tout brand qui veut scaler avec des ads. Les deux premiers sont soit non-fiables, soit réservés aux marques à $1M+/mois.

---

## 📌 Les 3 Types de CRO

### Type 1 — "Send It Strategy"
> *"Tu exécutes sans tester. Tu espères que ça marche mieux."*

- **Principe** : Faire des changements basés sur ce qui semble logique, sans validation
- **Risque** : L'exécution est plus souvent le problème que la stratégie (ex : reviews qui s'affichent sur les mauvais produits = CVR en baisse)
- **Recommandé pour** : Débutants avec un site vide, ou rebrands avec une équipe UI/UX expérimentée (ex : Audit.co)
- **Problème** : Aucune métrique ne confirme que les changements fonctionnent vraiment

### Type 2 — Scientific Method (A/B Testing)
> *"Le vrai test statistique. Sérieux, mais lent et coûteux."*

- **Principe** : Split testing rigoureux (page A vs page B), stat sig + stat power
- **Seuil minimum** : 300 à 1 000 orders par variant — sinon aucune confiance
- **Coût** : $10K à $30K/mois (stratégiste + copywriter + designer + dev + QA + account manager)
- **Durée** : 2 à 4 semaines minimum par test
- **Problème critique** : Spencer avait $25K/mois de payroll CRO et les résultats "sur papier" ne se retrouvaient pas dans les revenus réels
- **Exemple réel** : Test AA (page identique vs identique) montrant +10% CVR et +$20 AOV sur 8 000 visiteurs → **les deux pages étaient identiques** — c'est du bruit statistique
- **Recommandé pour** : Brands à $1M+/mois OU 300-500 orders/jour minimum

### Type 3 — Ad-Focused CRO ⭐ LA STRATÉGIE RECOMMANDÉE
> *"Le money printer. C'est tout ce qui m'intéresse."*

- **Principe** : Tester des landing pages et des offres **spécifiquement pour les ads**
- **Validation** : Pas besoin de stat sig — un bon ROAS ou CPA suffit pour scaler immédiatement
- **Vitesse** : Si la page génère 4.5x ROAS vs 1x avant → scale maintenant, pas dans 2 semaines
- **Recommandé pour** : **Tous les niveaux**, du débutant au brand établi

## 💡 Pourquoi Ad-Focused CRO Gagne

| Critère | Scientific | Ad-Focused |
|---------|-----------|------------|
| Vitesse | 2-4 semaines par test | Immédiat |
| Coût | $10-30K/mois | Faible |
| Validation | Stat sig requise | ROAS suffisant |
| Applicable à | $1M+/mois | Tout niveau |
| Impact visible | Sur papier | Dans le compte bancaire |

---

## 🏪 Application directe drinknellio.com

- **Ne pas A/B tester scientifiquement** avant d'avoir 300+ orders/jour — trop tôt, pas de données fiables
- **Stratégie à adopter maintenant** : Ad-Focused CRO → landing pages dédiées par angle/avatar pour Meta Ads
- Les grandes marques font exactement ça : Nellio UltraCalm doit avoir des **landing pages spécifiques par avatar** (stressed_professionals ≠ busy_moms ≠ students)
- Mesure = ROAS, pas CVR Shopify
- Outil mentionné pour la recherche concurrents : **Atria** (voir module CRO Research)
- Outil de split test mentionné : **IntelliGems** (pour quand on aura le volume)
- Outil référencé pour scientific CRO haut niveau : **splittesting.com** (Dylan Ander)
