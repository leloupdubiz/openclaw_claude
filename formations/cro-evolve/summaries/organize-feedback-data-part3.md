# TITRE: Customiser les Surveys — Aligner Questions & Réponses Multiselect

**CONCEPT CENTRAL**
Pour chaque client avec survey custom, adapter le template de feedback en synchronisant questions, réponses multiselect, et summaries pour éviter erreurs d'analyse AI.

**POINTS CLÉS**
- Les surveys custom nécessitent configuration manuelle (15-30 min)
- Utiliser les deux colonnes (question → options) pour chaque multiselect
- Copier/coller les colonnes existantes, puis updater question + options
- Vérifier que les options correspondent exactement à la survey original
- Ne PAS modifier les structures chart/summary initial — focus raw data + questions

**INSIGHTS APPLICABLES**
- Les misalignments question/options causent 40-60% d'erreurs AI
- Les surveys très customisées (5+ questions spéciales) prennent 3-4x plus long
- Garder reference à la survey original (VialSweep edit panel) pour vérifier réponses
- Les colonnes duplicate (copy/paste) économisent 70% du setup temps

**APPLICATIONS DRINKNELLIO.COM**
- Si survey Nellio custom : "Quelle transformation avez-vous ressentie?" → multiselect
- Ajouter colonne "Problème initial" et "Résultat après" = 2 colonnes options
- Documenter dans le sheet template : "Question alignée avec VialSweep 2024-04-01"
- Test avec un subset (20 réponses) avant import de 500+ entrants
