# TITRE — Module CRO : Test A-B sur Page Deck
## 🎯 Concept Central
Le test A-B est une méthode permettant de comparer deux versions d'une page web pour déterminer laquelle est la plus efficace. Dans ce cas, nous allons utiliser l'outil de test A-B de Page Deck pour comparer deux pages de landing.

## 📌 Points Clés
* Le test A-B est utilisé pour comparer deux versions d'une page web.
* L'outil de test A-B de Page Deck est simple et facile à utiliser.
* Il est nécessaire d'avoir Google Analytics 4 configuré pour utiliser l'outil de test A-B.
* L'outil de test A-B permet de créer des tests de redirection pour les pages de Page Deck.

## 💡 Insights Actionnables
