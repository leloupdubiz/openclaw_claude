# TITRE — Configuration de la Liste d'Entrants sur Klaviyo
## 🎯 Concept Central
La configuration de la liste d'entrants sur Klaviyo permet de gérer les inscriptions aux concours et de collecter les informations des participants de manière efficace.

## 📌 Points Clés
* Créer une nouvelle liste dans Klaviyo pour les entrants du concours
* Configurer la liste pour utiliser l'option de consentement unique (single opt-in) pour éviter les doubles validations
* Créer une clé API pour lier Klaviyo à l'outil de concours (Viral Sweep)
* Intégrer la clé API à l'outil de concours pour envoyer les informations des participants à Klaviyo
* Sélectionner la liste de destination pour les participants dans Klaviyo

## 💡 Insights Actionnables
* Utiliser l'option de consentement unique pour simplifier le processus d'inscription et éviter les doubles validations
* Créer une clé API dédiée pour chaque compte Klaviyo pour garantir la sécurité et la flexibilité
* Utiliser l'intégration pour collecter les informations des participants et les ajouter à la liste de destination dans Klaviyo

## 🏪 Applications drinknellio.com
Cette configuration peut être appliquée à différents scénarios de concours et de collecte de données, tels que les concours de produits, les jeux-concours, les campagnes de marketing, etc.
