# How To Organize Sheet With Feedback Data — Part 3 (Custom Surveys)

## Concept Central
Quand surveys sont très customisées (pas template standard), tu dois adapter dynamiquement le sheet. Multiple choice ≠ written answers. Chaque type a sa structure.

## Points Clés
1. **Types de questions** :
   - Multiple choice = colonnes options + filtering
   - Written answers (2 bonus entries) = données texte brutes
   - Chaque type a sa colonne + summary strategy

2. **Adaptation template personnalisé** :
   - Questions varient par client = adapter structure
   - Vérifier chaque réponse multiple choice
   - Localiser written answers (2 bonus entries)
   - Copier réponses exactes depuis VialSweep Edit

3. **Dupliquer colonnes pour aligner** :
   - Si MC et MC adjacents : dupliquer structure des 2 colonnes
   - Copy 2 colonnes de template → paste et update questions/réponses
   - Drag down et filter pour auto-count

4. **Utiliser filtering pour vérifier** :
   - Filter par réponse (ex: Travel Bag) → shows count
   - Si count vide = spelling ou range incorrecte
   - Corriger la formule ou extend range

5. **Charger summaries pour written answers** :
   - Questions MC = pas besoin summary (déjà counting)
   - Written answers = upload à ChatGPT pour summarization
   - Renommer section : "What do you want to see improved" → actionable title

## Insights
- Customisation ≠ disaster si on suit step-by-step
- Filtering = validation en temps réel
- Raw template = insurance contre les erreurs

## Applications drinknellio.com
- Si custom survey Nellio : identifier questions MC vs written
- Aligner chaque réponse exactement
- Charger written answers to ChatGPT (next step)
- Utiliser filtering pour verifier counts
