# How To Organize Sheet With Feedback Data - Part 3

## Concept Central
**Customisation de surveys = adaptation du sheet**. Chaque client ajoute ses questions → chaque question = nouvelle colonne à organiser.

## Points Clés
- **Certains clients modifient le survey** : ajoutent questions, suppriment questions, réordonnent
- **Solution** : dupliquer colonnes template existantes (ex: multiple choice) + adapter
- **Exemple Warroad** : ajout questions "age group", "product category interest" → nouvelles colonnes optionnelles
- Vérifier dans VialSweep l'ordre final des questions vs. le sheet
- **Quand reorder** : bouger colonnes (click-hold-drag) pour aligner avec survey réel
- **Quand ajouter** : copier une colonne template (ex: multiple choice filter) + adapter réponses

## Insights CRO
Plus le survey est custom, plus il faut de QA. Une colonne mal organisée = filter count vaut zéro pour cette option.
L'organisation de sheet = travail invisible mais **critique** pour résumés IA.

## Applications drinknellio.com

   - Copier 2 colonnes d'une question existing (ex: "How did you hear")
