# How To Summarize All Data & Create a Custom GPT

## Concept Central
**Master Custom GPT = single source of truth** sur tous tes customers. Upload tous les feedbacks (website + purchase + survey) → 1 GPT pour répondre n'importe quelle question sur ton audience.

## Points Clés
- **Fichiers multiples = meilleur contexte** pour GPT
  - Website feedback data (PDFs)
  - Purchase inspiration data (PDFs)
  - Custom survey data (PDFs)
  - Tout dans un seul GPT = la meilleure approche
- **Upload order** : peu importe, GPT va tout ingérer
- **Questions écrites uniquement** : multiple choice n'a pas besoin de summarization
- **Client logo** = nice touch (custom GPT shared avec client)
- **Conversation starters** = boutons de prompts template pour client
  - "What products do customers want to see next?"
  - "What should we improve on the store?"
  - "What do customers tell friends about us?"
- **Shareable link** : anyone (team, client) peut utiliser sans ChatGPT account
- **Bonus** : GPT peut générer ads visuels en demandant "Create a social media ad based on customer feedback"

## Insights CRO
1 GPT = toute la customer intelligence. Client peut poser des questions comme s'il parlait à un expert qui a lu 10,000 avis.
GPT synthétise = patterns + frequencies + specific quotes = insights actionnables.
