# M08-02 — Shopify Discount Codes Explained

## 🎯 Concept Central
Les codes de réduction Shopify sont un outil puissant pour stimuler les ventes et la croissance de votre entreprise. Il existe trois types de codes de réduction principaux : les codes de réduction manuels, les codes de réduction automatiques et les marques de réduction manuelles.

## 📌 Points Clés
- Les codes de réduction manuels sont utilisés pour les ventes secrètes ou les promotions ciblées.
- Les codes de réduction automatiques sont idéaux pour les ventes générales et s'appliquent automatiquement au panier.
- Les marques de réduction manuelles sont utilisées pour les produits spécifiques et nécessitent une mise à jour manuelle des prix.
- Les codes de réduction peuvent être combinés avec d'autres promotions ou frais d'expédition.
- Il est important de considérer les conséquences de l'utilisation de codes de réduction, tels que la perte de revenus potentiels si les clients n'avaient pas besoin d'un code pour acheter.

## ⚡ Insights Actionnables
- Utilisez des codes de réduction manuels pour les campagnes ciblées et les ventes secrètes.
- Mettez en place des codes de réduction automatiques pour les ventes générales et les promotions de grande envergure.
- Utilisez des marques de réduction manuelles pour les produits spécifiques ou les sections de clearance.
- Combinez les codes de réduction avec d'autres promotions pour maximiser les ventes.
- Évaluez soigneusement les conséquences de l'utilisation de codes de réduction sur vos revenus et votre stratégie commerciale.
