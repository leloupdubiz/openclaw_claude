# TITRE — Module CRO : Test de Livraison Intelligems
## 🎯 Concept Central
Le concept central de cette vidéo est de présenter la configuration d'un test de livraison pour Warroad en utilisant la plateforme Intelligems. L'objectif est de comprendre comment configurer un test de livraison et de mettre en évidence les points clés à considérer pour les tests futurs.

## 📌 Points Clés
- Le test de livraison est configuré pour tester différents tarifs de livraison, notamment la livraison standard et la livraison gratuite.
- Les tarifs de livraison peuvent être modifiés en cliquant sur "Preview Rates" et en sélectionnant les options de tarifs disponibles.
- Le seuil de livraison gratuite peut être défini pour chaque variante de test.
- Les noms des tarifs de livraison sont visibles par les clients et doivent être configurés de manière à être clairs et précis.
- Les modifications apportées à la page de panier, telles que le libellé de la livraison gratuite, nécessitent une coordination avec l'équipe Intelligems ou les développeurs.

## 💡 Insights Actionnables
- Assurez-vous de supprimer tous les produits du panier avant de configurer le test de livraison pour éviter les erreurs.
- Les modifications apportées aux libellés de livraison dans le panier nécessitent une coordination avec l'équipe Intelligems ou les développeurs.
- Les noms des tarifs de livraison doivent être configurés de manière à être clairs et précis pour les clients.
