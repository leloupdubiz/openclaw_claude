# 28. Comment télécharger une base de connaissances dans Chat GPT

## Résumé
Cette formation explique comment créer un Chat GPT personnalisé pour résumer les données de feedback des clients. Elle présente les étapes nécessaires pour télécharger les données dans Chat GPT.

## Points Clés
- Créer un compte Chat GPT payant pour accéder à ses fonctionnalités avancées
- Copier les données de feedback des clients et les coller dans un nouveau sheet
- Télécharger les données sous forme de PDF et les renommer pour une meilleure organisation
- Charger les fichiers dans Chat GPT pour permettre au modèle de les résumer
- Configurer le modèle pour qu'il puisse répondre aux questions sur les données téléchargées

## Actions Immédiates
- Créer un compte Chat GPT payant et accéder à la section "My GPTs" pour commencer la configuration
- Préparer les données de feedback des clients en les copiant et en les collant dans un nouveau sheet
- Télécharger les données sous forme de PDF et les charger dans Chat GPT pour commencer la résumé des données
