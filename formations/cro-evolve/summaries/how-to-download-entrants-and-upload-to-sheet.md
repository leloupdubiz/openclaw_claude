# How To Download Entrants & Upload To Sheet

## Concept Central
**VialSweep → Google Sheet** : pipeline d'extraction des données giveaway (emails + réponses) pour **alimenter la machine CRO** (feedback surveys).

## Points Clés
- **VialSweep** : télécharger les entrées (Entrances tab) après fin du giveaway
- Fichier volumineux ? → Export par email (notification VialSweep)
- **Google Sheet** : créer raw file (sauvegarde) + clean file (travail)
- **Nettoyage** : supprimer colonnes inutiles (IP, location) jusqu'aux prénoms/noms
- Structure finale = emails + first/last name + réponses de survey

## Insights CRO
Le giveaway = **acquisition + feedback** d'un coup. Les données des entrants = avatars réels, pain points réels.
Bien organiser = summarize plus tard sans friction.

## Applications drinknellio.com
1. **Lancer un giveaway Nellio** (VialSweep) pour recruter audience DE
2. **Survey questions critiques** :
   - "Comment as-tu découvert Nellio ?" (attribution)
   - "Quel problème Nellio résout-il pour toi ?" (desires mapping)
   - "Qu'est-ce qui pourrait l'améliorer ?" (pain points)
3. **Dataset clean** → Jasper/GPT summarization → insights avatars
