# Key Metrics For CRO — Résumé CRO

## 🎯 Concept Central
3 métriques fondamentales définissent la santé d'une boutique DTC : **Conversion Rate (CVR), Average Order Value (AOV), Revenue Per Session (RPS)**. À partir de ces 2, tu calcules le **Shopify Score** = ton principal diagnostic tool pour savoir où optimiser.

## 📌 Points Clés
- **Core metrics definitions**
  - **CVR (Conversion Rate)** : % de visitors qui achètent
  - **AOV (Average Order Value)** : prix moyen par commande
  - **RPS (Revenue Per Session)** : fonction de CVR × AOV
  
- **Shopify Score = CVR × AOV**
  - 0-100 : Produit ne communique pas value OU trafic horrible (bot/lowball)
  - 100-250 : Quelque chose fonctionne mais pas assez pour scale
  - 250-400 : Zone d'or = commencer à scale le trafic
  - 400+ : Focus sur ads (store can handle the traffic)

- **Add-to-Cart Ratio** (métrique secondaire)
  - Ratio ATC:CVR entre 3:1 et 2:1 = healthy
  - 4:1+ = issue post-ATC (checkout, shipping, upsells)
  - 2:1 = elite (rare)
  - 1:1 = jamais vu (théorique ideal)

- **Margin ≠ Score** (importante nuance)
  - High margin (70-85%) ≠ high score
  - Low margin (50-55%) peut avoir score plus élevé
  - Trade-off : price vs. conversion rate

## 💡 Insights Appliqués

| Shopify Score | Action Recommandée |
|---------------|-------------------|
| < 100 | Audit produit, pricing, website health |
| 100-250 | Optimize site + pages aggressively |
| 250-400 | Scale trafic + continue optimization |
| 400+ | Focus 100% sur creative + media buying |

## 🍹 Applications drinknellio.com

1. **Calculer Nellio Score actuel**
   - Assumption CVR : ~1.5% (guesstimate sans data)
   - Assumption AOV : €35-45 (poudre single + potential upsell)
   - Nellio Score = 1.5 × 40 = **60** (zone 0-100)
   - **Action** : Audit du PDP, pricing validation, trafic quality check

2. **Benchmark after optimization**
   - Post-CRO : target CVR = 2-3%
   - Post-upsell : target AOV = €50-65
   - Target Nellio Score = **150-195** (zone 100-250)
   - Then: scale the ads

3. **Add-to-Cart monitoring**
   - Current ratio unknown (need Meta pixel data)
   - Target : 3:1 ou better
   - Si worse: checkout flow, shipping price, upsell issue

4. **ROAS formula** (bonus)
   - ROAS = CVR × AOV × CTR ÷ Cost Per Click
   - To improve : focus CVR + AOV (page level) + CTR + media (ad level)
   - **CRO = domaine de CVR + AOV**
