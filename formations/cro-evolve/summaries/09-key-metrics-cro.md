# Key Metrics For CRO

## Concept Central
**Shopify Score (CR × AOV) = santé globale du store**. C'est la métrique mère qui détermine si on optimize la page (0-250), scale ads (250+), ou change stratégie produit (<100). Tous les autres KPIs (add-to-cart, ROAS) en découle.

## Points Clés

### Core Metrics Definitions
- **CVR/CR** : conversion rate (% sessions → purchases)
- **AOV** : average order value (revenu moyen par commande)
- **RPS** : revenue per session = CVR × AOV

### Shopify Score Breakdown (Conversion Rate × AOV)
| Plage | Santé | Action |
|-------|-------|--------|
| 0-100 | 🔴 Critique | Fix product, pricing, ou trafic (bot/quality issue) |
| 100-250 | 🟡 Développement | Optimize pages (site = store/landing) |
| 250-400 | 🟢 Bon | Scale traffic + continuer optimisation |
| 400+ | 🟢 Elite | Saturate ads, focus créative (store = cash cow) |

### Add-to-Cart Ratio
- **Elite** : 2:1 (6% ATC, 3% CVR)
- **Good** : 3:1 (6% ATC, 2% CVR)
- **Warning** : 4:1+ = checkout friction ou trafic non-qualified
- **Insight** : utilisé pour diagnostiquer post-click issues (shipping, upsells, etc)

### ROAS Equation
```
ROAS ↑ = (CVR × AOV × CTR) ↑
```
Améliorer CVR ou AOV = améliorer ROAS + permettre scaling.

## Insights
Shopify Score absout : 250+ = scaling zone. Margin paradoxe : margins ultra-hautes (96%) souvent = mauvais pricing (trop élevé = basse CVR). Sweet spot = balance prix/marge/conversion.

## Applications drinknellio.com
- **Nellio baseline** : calculer Shopify Score actuel (données Meta)
- **Target Shopify Score** : viser 300+ avant scaling (250 = minimum)
- **AOV optimization** : bundle offers, upsells (post-purchase email)
- **CVR targets** : 2-3% initial (poudre drinkable, marché DE)
- **Add-to-Cart monitoring** : si >4:1, audit checkout Shopify (shipping, form friction)
- **ROAS focus** : créative + page = levier principal avant scaling (plutôt que baisser COGS)
