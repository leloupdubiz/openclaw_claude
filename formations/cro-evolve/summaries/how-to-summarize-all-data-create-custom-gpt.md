# How to Summarize All Data & Create a Custom GPT

## Concept Central
Créer un GPT customer-obsessed pour chaque client, entraîné sur **toutes** les données de feedback (site web + survey responses), et le partager avec l'équipe/client pour l'analyse en temps réel.

## Points Clés
- **Étape 1 : Upload ALL knowledge bases** :
  - Website feedback data (écrit)
  - Giveaway survey responses (écrit)
  - Customer reviews (optionnel)
  - Competitive research (optionnel)
  - → Uploader TOUT à la fois = plus efficace que itérer question par question
- **Étape 2 : Custom GPT setup** :
  1. Ajouter le logo du client (visual touch)
  2. Écrire 4-5 "conversation starters" (quick prompts)
     - "What products do customers want to see next?"
     - "What do customers tell others about our products?"
     - "Based on feedback, what ads would convert new customers?"
  3. Partager le lien avec l'équipe/client
- **Étape 3 : Utilisation** :
  - N'importe qui peut cliquer sur les conversation starters
  - Poser des questions custom ("What's the #1 objection?")
  - GPT génère des résumés avec citations du feedback
  - Peut même générer des idées créatives ad
- **Bonus** : Le GPT peut générer des visual ad concepts (avec images)

## Insights
Un GPT client bien entraîné = tool de décision 10x plus rapide que lire 500 réponses manuellement. C'est le contraire de "generic advice".

## Applications drinknellio.com
