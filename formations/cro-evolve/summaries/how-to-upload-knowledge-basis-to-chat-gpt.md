# How To Upload Knowledge Basis To Chat GPT

## Concept Central
**Custom GPT = machine à résumer feedback**. Upload les data brutes (PDFs exported du sheet) → GPT va synthesizer automatiquement les réponses.

## Points Clés
- **ChatGPT PRO** ($20/mois) = créer Custom GPTs
- **Créer Custom GPT** : Profile → My GPTs → Create new
- **Configuration** : nom client + description simple + instructions (prompt template fourni)
- **Upload files** : exporter chaque question écrite du sheet en PDF
  - "What should we improve ?" → export juste cette colonne → PDF → upload dans GPT
- **Nom fichier = important** : "warroad-improve-store.pdf" aide GPT à contextualize
- **Un GPT = un client** : tous les PDFs de ses questions dedans
- **Shareable** : copier link + envoyer à team/client

## Insights CRO
Au lieu de lire 1000 réponses, GPT les **synthétise par fréquence** : "35% mentioned ..., 22% complained about ...".
Temps : 2h de lecture humaine → 30 secondes GPT.
Qualité : si bien organisé, très précis.

## Applications drinknellio.com
1. **Créer Custom GPT "Nellio Feedback Analyzer"**
   - Nom : "Nellio UltraCalm Customer Insights"
   - Upload 2 PDFs (questions écrites) :
     - nellio-what-problem-solved.pdf
     - nellio-what-to-improve.pdf
2. **Instructions template** :
   ```
   Analyze customer feedback on Nellio UltraCalm.
   Summarize by frequency. Include specific quotes.
   ```
3. **Share link avec team** → analyser trends automatiquement
