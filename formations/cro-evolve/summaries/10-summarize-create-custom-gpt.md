# How To Summarize All Data & Create a Custom GPT — Full Workflow

## Concept Central
Pipeline complète : sheet nettoyé → extraire written answers → ChatGPT summarize → créer Custom GPT pour team. Dernière étape = asset réutilisable client-centric.

## Points Clés
1. **Décision : Full data vs partial ?** :
   - Upload TOUS les fichiers à GPT vs 1 par 1
   - Résultat : peu de différence, mais full data = plus contexte
   - Recommendation : full batch (moins de temps, plus de contexte)

2. **Setup Custom GPT Final** :
   - ChatGPT+ → Create Custom GPT
   - Name: "[Client] Customer Insights & Feedback"
   - Upload tous les PDF files (written answers)
   - Configuration : toggle "Anyone with link can use"

3. **Adding Conversation Starters** :
   - Edit GPT > Add conversation starters
   - Exemples pour Warroad :
     - "What products do customers want to see next?"
     - "What do customers want improved on store?"
     - "What types of ads would attract new customers?"
     - "What do customers tell friends about our products?"
   - Chaque starter = basé sur une question du survey

4. **Custom GPT Features** :
   - Team can chat avec le GPT sans refaire le work
   - Client peut accéder via lien (shareable)
   - GPT génère visuels + ads concepts + copy suggestions
   - Trained sur TOUS customer feedback = customer-obsessed

5. **Workflow en Practice** :
   - Ask GPT : "What products do customers want?" 
   - GPT references all data, returns by frequency + specifics
   - Peut générer visual ad concepts directement (!)
   - Peut créer social media copy basé sur data

6. **Output Examples** :
   - "Safety and cut prevention strong demand... Michael silver threading, innovative base layers..."
   - Visual concept: Product safety positioning
   - Text: Engagement copy pour social

7. **Sharing & Scalability** :
   - Copy GPT link, send to team/client
   - Multiple people can use simultaneously
   - Asset living (add more data over time)

## Insights
- Full-data GPT = 10x better context than manual analysis
- Conversation starters lower activation energy (client can self-serve)
- GPT can generate ideas (ads, copy, visuals) from pure customer data
- Asset réutilisable = ROI leverage pour team
