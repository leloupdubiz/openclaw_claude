# TITRE — Module CRO
## 🎯 Concept Central
Le concept central de cette vidéo est de configurer les fondations d'un site Web à l'aide de l'outil PageDeck. L'objectif est de créer un site Web fonctionnel avec des paramètres de base tels que la connexion d'un domaine personnalisé, la configuration des réglages du site, l'ajout de polices de caractères et la connexion de Google Analytics.

## 📌 Points Clés
- Configurer un domaine personnalisé avec PageDeck
- Comprendre les réglages du site et les thèmes
- Ajouter des polices de caractères personnalisées
- Connecter Google Analytics pour suivre les performances du site
- Ajouter des scripts de suivi pour des outils tels que TripleWhale, Heatmap.com et Meta Pixel

## 💡 Insights Actionnables
- Utiliser le code EZA15 pour obtenir 15% de réduction sur l'abonnement à PageDeck
- Configurer correctement les paramètres DNS pour connecter un domaine personnalisé
- Utiliser des outils tels que What Font pour identifier les polices de caractères utilisées sur un site Web
- Comprendre l'importance de sauvegarder les modifications apportées aux réglages du site

## 🏪 Applications drinknellio.com
Cette vidéo n'est pas spécifiquement liée à drinknellio.com, mais les compétences et les connaissances acquises peuvent être appliquées à la configuration et à l'optimisation de n'importe quel site Web, y compris ceux créés avec des outils comme PageDeck.

## ⚡ Citation Clé
"Assurez-vous de sauvegarder vos modifications après avoir apporté des changements aux paramètres du site, car oublier de le faire peut entraîner des problèmes."
