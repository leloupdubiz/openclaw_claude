# TITRE — Module CRO
## 🎯 Concept Central
L'exécution de tests gagnants est un élément clé de l'optimisation de l'expérience utilisateur et de la conversion. Dans cette vidéo, nous allons explorer les différentes méthodes pour exécuter des tests, notamment les tests de redirection, les éditions sur site, les tests de modèles et les tests de thèmes.

## 📌 Points Clés
* Les tests de redirection : il s'agit de tester différentes URL pour voir laquelle fonctionne le mieux.
* Les éditions sur site : il s'agit de tester différentes éditions d'une page pour voir laquelle fonctionne le mieux.
* Les tests de modèles : il s'agit de tester différents modèles de page pour voir lequel fonctionne le mieux.
* Les tests de thèmes : il s'agit de tester différents thèmes pour voir lequel fonctionne le mieux.

## 💡 Insights Actionnables
* Pour exécuter un test de redirection, il suffit de créer une redirection URL dans Shopify.
* Pour exécuter un test d'édition sur site, il suffit de modifier la page en question et de sauvegarder les changements.
* Pour exécuter un test de modèle, il suffit de créer un nouveau modèle et de l'attribuer à la page en question.
* Pour exécuter un test de thème, il suffit de créer un nouveau thème et de l'attribuer à la page en question.
