# Summariser Données & Créer Custom GPT Complet — Full Stack AI

## Concept Central
Évolution du module 7 : au lieu de 1 GPT par question, créer 1 GPT master avec **tous** les fichiers knowledge, conversations starters, et sharing ability. Client-obsessed AI assistant pour toute la team.

## Points Clés
- **Single vs Multiple Files Approach**
  - **Single file per question** : Summaries précises mais laborieux (N questions = N PDF uploads)
  - **All files at once** : Summaries slightly varient mais plus efficace, moins time-intensive
  - Recommendation : Upload all files, run GPT plusieurs fois pour variantes, garder best summary
- **GPT Master Setup**
  1. Upload ALL written feedback PDFs (ex: 4-6 fichiers pour template)
  2. Set Name: "[Client Name] Customer Insights" (ex: "Warroad Customer Insights")
  3. Instructions: Standard prompt (référence sheet fournie)
  4. Add Client Logo (visual polish)
- **Conversation Starters** (Critical for UX)
  - "What products do customers want to see next?"
  - "What improvements do customers request most?"
  - "What pain points drive purchases?"
  - "Based on feedback, what ads would resonate?"
  - Base starters sur questions custom du survey
- **Sharing & Distribution**
  - Click "Share" → get public link
  - Send to clients, team, stakeholders
  - Anyone can query without ChatGPT Pro subscription (major wow factor)
- **Advanced Use** : Ask GPT to generate ad copy, visual concepts, product ideas based on all customer data

## Insights Professionnels
- **Leverage** : Investir $20/mois ChatGPT Pro × indexing = infinite customer insight queries for team
- **Client Engagement** : Sharing GPT with client = high-touch service, more stickiness
