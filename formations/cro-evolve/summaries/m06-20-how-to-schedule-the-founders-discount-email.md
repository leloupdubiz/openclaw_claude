# TITRE — Module CRO : Comment planifier l'e-mail de réduction du fondateur

## 🎯 Concept Central
Le concept central de cette vidéo est de planifier et de segmenter l'e-mail de réduction du fondateur, ainsi que de créer un code de réduction pour les participants à un concours qui n'ont pas gagné.

## 📌 Points Clés
- Segmentation de la liste d'e-mails pour exclure les gagnants du concours
- Création d'un nouveau code de réduction pour les participants non gagnants
- Planification de l'e-mail de réduction du fondateur pour être envoyé après l'annonce des gagnants
- Test du code de réduction pour s'assurer qu'il fonctionne correctement

## 💡 Insights Actionnables
- Assurez-vous de segmenter votre liste d'e-mails pour cibler les participants appropriés
- Créez un code de réduction unique pour les participants non gagnants
- Planifiez l'e-mail de réduction du fondateur pour maximiser l'impact
- Testez toujours le code de réduction avant de l'envoyer aux participants

## 🏪 Applications drinknellio.com
Cette vidéo peut être appliquée à drinknellio.com en créant des campagnes d'e-mail ciblées pour les participants à des concours ou des promotions, en offrant des codes de réduction exclusifs pour les non-gagnants, et en planifiant les e-mails pour maximiser l'engagement et les ventes.
