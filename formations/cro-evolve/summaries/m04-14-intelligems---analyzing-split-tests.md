# TITRE — Module CRO : Analyse de Tests dans IntelliGems

## 🎯 Concept Central
L'analyse de tests dans IntelliGems est une étape cruciale pour comprendre les performances de vos variantes de test et prendre des décisions éclairées pour améliorer vos campagnes de marketing.

## 📌 Points Clés
- **Durée des tests** : Il est essentiel de laisser les tests s'exécuter suffisamment longtemps pour collecter des données significatives. IntelliGems recommande d'atteindre au moins 300 commandes par variante.
- **Analyse des données** : IntelliGems offre diverses options de filtrage, notamment par appareil (desktop vs mobile), type de client (nouveau vs récurrent), et source de trafic.
- **Quadrants clés** : Les quadrants clés à analyser incluent desktop vs mobile et nouveau vs récurrent.
- **Statistiques** : Comprendre les statistiques, comme la signification statistique, est crucial pour interpréter les résultats des tests de manière fiable.

## 💡 Insights Actionnables
- **Utiliser les filtres** : Utilisez les options de filtrage pour affiner vos analyses et comprendre mieux les performances de vos variantes de test.
- **Surveiller les tendances** : Le graphique de tendance est utile pour observer l'évolution des performances de vos tests au fil du temps.
- **Réévaluer les tests** : Soyez prêt à ajuster ou à arrêter les tests en fonction des résultats et des objectifs de votre campagne.

## 🏪 Applications drinknellio.com
IntelliGems offre une plateforme conviviale pour créer, gérer et analyser des tests A/B. Les fonctionnalités incluent la possibilité de créer des groupes de test, de configurer des filtres pour les données, et d'afficher les résultats sous forme de graphiques et de tableaux pour une meilleure compréhension.

## ⚡ Citation Clé
"La clé pour une analyse efficace est de collecter suffisamment de données pour prendre des décisions éclairées. IntelliGems facilite ce processus avec ses outils d'analyse et de filtrage."
