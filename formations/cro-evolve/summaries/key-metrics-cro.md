# Key Metrics For CRO

## Concept Central
Shopper Score (Conversion Rate × AOV) = ta clé pour savoir si tu dois optimiser le site ou les ads — sans ce benchmark, tu optimises les mauvaises choses.

## Points Clés
- **Conversion Rate (CVR)** : sessions → purchases. Métrique de site health
- **Average Order Value (AOV)** : revenue ÷ transactions. Métrique de pricing + upsell
- **Revenue Per Session (RPS)** : CVR × AOV. La seule métrique qui compte vraiment
- **Shopify Score** : (CVR × AOV) × 100. Ranges : 0-100 (broken), 100-250 (optimize), 250-400 (scale), 400+ (ads-only)
- **Add-to-Cart Ratio** : 4:1 (mauvais) → 2:1 (elite). Signal de funnel leak post-add-to-cart

## Insights
**Shopify Score ≤100** : ton produit ou pricing est cassé. Aucun ad spend ne salvera ça.
**Score 100-250** : tu convertis, mais pas assez pour scale ads. Focus CRO, pas ads.
**Score 250-400** : tu es prêt. Scale les ads, ça va marcher.
**Score 400+** : ton site gère la charge. Le problème n'est PAS ton site, c'est tes ads. Dépense 100x plus sur creative.

Exemple : Client à 4.23% CVR + $152 AOV (score ~640) = $220K/mo après 45j de scaling ads. Même budget sur client avec CVR 1% + $50 AOV = perte de temps.
