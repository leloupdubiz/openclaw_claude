# How To Upload Knowledge Basis To Chat GPT

## Concept Central
**Custom GPT = moteur de recherche client-spécifique**. En uploadant les données de feedback dans une custom GPT ChatGPT ($20/mois), on crée une IA "entraînée" sur les clients qui peut répondre à n'importe quelle question sur leur business, leurs désirs et leurs objections.

## Points Clés
- **ChatGPT+ obligatoire** : $20/mois pour custom GPT
- **Custom GPT création** : Profile → My GPTs → Create → Configure
- **Upload fichiers** : copier chaque colonne written answer (bonus entries) en PDF
- **Naming PDF** : utiliser le nom de la question (ex: "improved-store-pmd.pdf")
- **Instructions simple** : prompt prédéfini (voir loom)
- **File preparation** : copier réponses écrites uniquement (pas les multiple choice) → nouvelle sheet → file → download PDF
- **Prompt usage** : copier la question depuis sheet → coller dans GPT → générer résumé → copier résumé → coller dans sheet
- **Reusable** : le custom GPT reste accessible, peut être partagé (link sharing)
- **Logo client** : optional mais good practice (reconnaissance)

## Insights
Custom GPT remplace Jasper pour cette pipeline. Avantage : interface unifiée, partage facile, coût 1/10eme de Jasper. Limitation : processing lent si beaucoup données, nécessite tuning du prompt parfois.
