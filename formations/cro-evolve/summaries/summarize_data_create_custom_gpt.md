# Résumé : Résumer Données & Créer Custom GPT (How To Summarize All Data & Create a Custom GPT)

## Concept Central
Le Custom GPT finale est la pièce maîtresse : entraîner une IA sur TOUTES les données client (website feedback + purchase data + reviews + market research) crée un "customer obsessed AI" qui peut répondre à toute question stratégique pour le client. C'est le "single source of truth" pour customer insights.

## Points Clés
- **Custom GPT multi-source** :
  - Upload TOUTES les données : feedback survey + website reviews + purchase data + competitor data
  - ChatGPT index les fichiers automatiquement
  - Une seule query = réponse basée sur tous les data sources

- **Avantages du multi-source vs single source** :
  - Plus de nuance (feedback + reviews parfois divergent)
  - Meilleur pattern recognition
  - Aucune différence majeure si données bien structurées (time investment est petit)

- **Workflow summary** :
  1. Créer custom GPT (configure, name, logo du client)
  2. Upload PDFs de feedback questions
  3. Upload website data / reviews / competitor insights
  4. Test queries
  5. Ajouter conversation starters (exemples de questions utiles)
  6. Share link avec team/client

- **Conversation starters** (examples) :
  - "What products do customers want to see next ?"
  - "What do customers tell others about our products ?"
  - "What types of ads would get new customers to purchase ?"
  - "What improvements do customers want on the store ?"

- **Bonus : AI-generated visuals** :
  - Le GPT peut générer ad concepts visuels basés sur feedback data
  - Prompt minimaliste ("Make an engaging interactive ad") = GPT crée full photo + copy

## Insights
- Multi-source = temps setup juste un peu plus long, mais ROI énorme (client a "God mode" knowledge)
- Les conversation starters = UX game-changer (clients cliquent plutôt que formuler query)
- Custom GPT with rich data = 10x better than generic business advice
