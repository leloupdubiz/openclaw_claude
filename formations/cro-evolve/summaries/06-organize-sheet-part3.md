# How To Organize Sheet With Feedback Data — Part 3

## Concept Central
**Customisation template = adaptation aux questions réelles du client**. Chaque giveaway a ses questions spécifiques ; le template doit refléter cette réalité pour que les résumés IA soient pertinents, pas génériques.

## Points Clés
- **Charts non-fiables** : ignorer les graphiques du template, focus sur données brutes + résumés
- **Colonnes multiples réponses** : copier-coller la structure "question + options" si réponses supplémentaires
- **Réponses written vs multiple choice** : traiter différemment — écrits = résumés IA, choix = comptage/filtrage
- **Drag-fill** : copier colonne, coller, puis drag poignée vers le bas pour appliquer formule (comptage réponses)
- **Ordre des questions** : doit correspondre au VialSweep survey setup
- **Bonus entries** : ces questions NÉCESSITENT les résumés Jasper/GPT (données écrites)
- **Suppression test data** : avant import GPT, nettoyer test entries/client emails

## Insights
Qualité du template = qualité résumés. Un décalage question-réponse = bruit systématique dans IA. Toujours comparer VialSweep survey setup avec clean sheet avant résumé.

## Applications drinknellio.com
- **Giveaway Nellio template** : modifier colonne "options" selon réponses réelles du sondage VialSweep
- **Bonus entries Nellio** : questions ouvertes (stress → bien-être ?) doivent aller dans résumés GPT
- **Vérification pré-résumé** : checklist ordre questions vs VialSweep avant importer dans Jasper
