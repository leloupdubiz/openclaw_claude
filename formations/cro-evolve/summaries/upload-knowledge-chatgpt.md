# How To Upload Knowledge Basis To Chat GPT

## Concept Central
Créer un Custom GPT entraîné sur data client = mini AI expert dédié. Upload documents (PDF, sheets, feedback) → GPT summarize + réutilisable infiniment.

## Points Clés
1. **Prérequis** :
   - ChatGPT Plus subscription ($20/mois)
   - Accès "My GPTs" (profile → My GPTs)
   - Documents prêts (PDF de feedback, website info, reviews, etc.)

2. **Setup GPT** :
   - Create new GPT
   - Configure : nom, description, instructions (simple prompt)
   - Click "Upload files" → PDF, sheets export
   - Test avec questions simples

3. **Extraction data pour upload** :
   - Copy questions-réponses ouvertes du sheet (pas MC)
   - Créer nouveau sheet "Export"
   - Copier juste les réponses texte
   - Exporter en PDF
   - Renommer : "[Question] - [ClientName]"

4. **Prompt template** :
   ```
   "You are a customer insights expert for [Brand].
   Analyze the customer feedback files provided.
   Summarize by frequency and importance.
   Be specific, cite actual customer words."
   ```

5. **Partage** :
   - Generate link → share avec client + team
   - Client peut poser questions directement au GPT
   - Conversation starters (buttons) pour questions pré-définies

## Insights
Custom GPT = alternative à Jasper moins chère et plus flexible. GPT "learns" contexte exact du client. Plus utile que un GPT generic.

One-time setup = multiple uses.

## Applications drinknellio.com
**Nellio Custom GPT workflow** :

1. **Setup** :
   - Créer GPT : "Nellio Customer Insights AI"
   - Description : "Knows everything about Nellio UltraCalm customers"
   - Instructions simples (voir template)

2. **Documents à uploader** :
   - Website copy + benefits drinknellio.com
   - Customer reviews (Amazon, site)
   - Giveaway feedback PDF (open-text responses)
   - Competitor reviews (Ashwagandha supps Amazon)

3. **Conversation starters** :
   ```
   "What do Nellio customers want to see improved?"
   "Which ingredients do customers care most about?"
   "What benefits do customers mention most?"
   "What objections come up about price/taste?"
   "What's the #1 use case for Nellio?"
   ```

4. **Use cases** :
   - Hook writer : "Generate 5 hooks based on customer feedback"
   - Script writer : "What objections should script address?"
   - Ad copywriter : "Top 3 customer pain points to address"
