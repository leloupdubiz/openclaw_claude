# TITRE — Module CRO
## 🎯 Concept Central
Le concept central de cette vidéo est la mise en place d'un test A-B visuel à l'aide d'IntelliGems, un outil de test et d'optimisation de site web. L'objectif est de comprendre comment créer et mettre en œuvre des tests visuels pour améliorer l'expérience utilisateur et augmenter les conversions.

## 📌 Points Clés
- **Cible spécifique** : Il est essentiel de cibler des pages spécifiques pour les tests A-B, en utilisant des options comme "contient" ou "égale à" pour s'assurer que le test est appliqué aux bonnes pages.
- **Utilisation d'IntelliGems** : L'outil permet de réaliser des tests visuels sans nécessité de compétences techniques approfondies, ce qui est particulièrement utile pour les tests de messagerie et d'images.
- **Éditeur visuel** : IntelliGems propose un éditeur visuel qui permet de modifier directement les éléments de la page web, tels que les textes et les images, pour créer des variantes de test.
- **Problèmes de compatibilité mobile** : Il peut y avoir des problèmes de compatibilité avec les appareils mobiles, nécessitant parfois des ajustements manuels ou l'aide d'un développeur.
- **Utilisation de codes personnalisés** : Pour les tests plus complexes, il est possible d'injecter du code personnalisé, mais cela nécessite des compétences en développement.

## 💡 Insights Actionnables
- **Sélectionnez soigneusement les pages cibles** pour vos tests A-B pour vous assurer que vous atteignez le public visé.
- **Utilisez l'éditeur visuel d'IntelliGems** pour des tests rapides et simples, comme les tests de headlines ou d'images.
- **Testez sur différents appareils** pour vous assurer que vos modifications sont compatibles avec les appareils mobiles.
- **N'hésitez pas à demander de l'aide** si vous rencontrez des problèmes de compatibilité ou si vous avez besoin d'injecter du code personnalisé.

## 🏪 Applications drinknellio.com
Cette vidéo et ses conseils peuvent être appliqués à n'importe quel site web qui utilise IntelliGems pour l'optimisation et les tests A-B, y compris drinknellio.com. Les principes de ciblage de pages spécifiques, d'utilisation de l'éditeur visuel, et de test sur différents appareils sont universellement applicables.
