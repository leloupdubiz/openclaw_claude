# Vibe Coding CodePlaybook #3

**Concept Central**
Le concept central de cette vidéo est de montrer comment créer un site web en utilisant l'intelligence artificielle (IA) et des outils de développement rapide tels que Cursor, ChatGBT et Astro. L'objectif est de créer un site web minimaliste et clean pour présenter un portfolio et permettre aux visiteurs de s'inscrire à une newsletter.

**Points Clés**

* Utilisation de l'IA pour créer un site web
* Choix de la pile technologique (tech stack) : Astro, Tailwind CSS, Cloudflare Pages
* Utilisation de Cursor comme éditeur de code
* Utilisation de ChatGBT pour obtenir des conseils et des recommandations
* Création d'un site web avec une page d'accueil, une page à propos et une page de projets
* Intégration d'un formulaire d'inscription à une newsletter
* Déploiement du site web sur Cloudflare Pages

**Insights Actionnables**

* Utiliser l'IA pour accélérer le processus de développement web
* Choisir les bons outils pour le développement web, tels que Cursor et ChatGBT
* Utiliser des frameworks tels que Astro et Tailwind CSS pour créer des sites web rapides et minimalistes
* Intégrer des formulaires d'inscription à des newsletters pour collecter des adresses e-mail
* Déployer des sites web sur des plateformes telles que Cloudflare Pages pour une mise en ligne rapide et sécurisée
