# Vibe Coding CodePlaybook #12

**Concept Central**
Le concept central de cette vidéo est l'introduction aux MCP (Model, Context, Protocol) servers, qui sont des intégrations standardisées pour connecter les outils d'intelligence artificielle (IA) aux différents outils et services en ligne. Les MCP servers permettent aux outils d'IA de se connecter à des bases de données, des services de paiement, des outils de développement, etc.

**Points Clés**

* Les MCP servers sont des intégrations standardisées pour connecter les outils d'IA aux différents outils et services en ligne.
* Il existe de nombreux MCP servers différents, chacun avec ses propres capacités et fonctionnalités.
* Les MCP servers peuvent être utilisés pour connecter les outils d'IA à des bases de données, des services de paiement, des outils de développement, etc.
* Les MCP servers peuvent être installés et configurés de manière différente en fonction de l'outil d'IA utilisé.
* Les MCP servers peuvent être utilisés pour automatiser des tâches, récupérer des données, effectuer des recherches, etc.

**Insights Actionnables**

* Utiliser les MCP servers pour connecter les outils d'IA aux différents outils et services en ligne peut améliorer l'efficacité et la productivité.
* Il est important de choisir les MCP servers appropriés pour les besoins spécifiques de votre projet ou de votre entreprise.
* Il est important de configurer et d'installer les MCP servers de manière correcte pour éviter les problèmes de sécurité et de performances.
* Les MCP servers peuvent être utilisés pour créer des workflows no-code/low-code pour automatiser des tâches et des processus.
