## Frameworks Clés Extraits

### 1. Desire-First Marketing
```
On ne vend pas un produit → On channel un desire existant
Desire Strength = Intensity × Scope
```

### 2. 5 Stages of Awareness
```
UNAWARE → PROBLEM → SOLUTION → PRODUCT → MOST AWARE
(Traffic chaud → froid)
```

### 3. Avatar Architecture
```
CORE AVATAR (1 desire)
├── Sub-Avatar 1 (behavior + pain) → Angle A
├── Sub-Avatar 2 (behavior + pain) → Angle B
└── Sub-Avatar 3 (behavior + pain) → Angle C
```

### 4. 3-2-2 Testing Method
```
3 Créatifs × 2 Body Copies × 2 Headlines = 12 ads/test
```

### 5. Marksman → Sniper → Scale
```
MARKSMAN: Test 3 angles différents → Trouver direction
SNIPER: Deep-dive sur winner → 3 variations
SCALE: Augmenter budget → Nouvelles audiences
```

## Notes Personnelles

### Ce qui m'a marqué dans EVOLVE :

1. **Concept vs Angle** - La distinction la plus importante. La plupart des marketers confondent. Concept = ce qu'on teste. Angle = raison d'achat du client.

2. **Sub-avatar System** - Très smart. Au lieu de personas génériques, on va chercher les behaviors spécifiques qui mènent à des angles actionnables.

3. **Marksman vs Sniper** - Bonne métaphore. Tester large d'abord, puis sniper quand on a trouvé la direction.

4. **3-2-2 Method** - Structure de test claire. 3 variations = assez pour apprendre, pas trop pour se disperser.

5. **Hook-first** - 3 secondes pour capturer. Tout le reste dépend du hook.

### Ce que j'ai appris :

- La formation est très complète et actionnable
- L'approche "desire-first" est contre-intuitive mais logique
- Le système sub-avatar → angle est puissant
- Les 5 awareness levels sont sous-utilisés par la plupart
