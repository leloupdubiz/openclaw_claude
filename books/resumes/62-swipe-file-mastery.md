- **Priorité :** 🟡 MEDIUM

### 🎯 CONCEPT CENTRAL
> "Swipe files ne sont pas pour copier. Ils sont pour comprendre les patterns et s'inspirer."

Guide pour créer et utiliser un swipe file efficace — collection d'ads, emails, pages qui convertissent.

### 📚 CONTENU DÉTAILLÉ

#### SECTION 1 : SWIPE FILE ORGANIZATION
**Structurer ta collection :**
- By format (Facebook ads, landing pages, emails, VSLs)
- By angle/problem (pain point, desire, fear)
- By awareness stage (unaware → most aware)
- By sophistication level (market maturity)
- By industry/niche (health, wealth, relationships)

#### SECTION 2 : WHAT TO SWIPE
**Collecter intelligemment :**
- Ads qui apparaissent plusieurs fois (proven winners)
- Landing pages avec preuves sociales fortes
- Email subject lines (open rates élevés)
- Hooks qui capturent attention en 3 sec
- Offers avec guarantees créatives

#### SECTION 3 : ANALYSIS FRAMEWORK
**Analyser, pas juste collecter :**
- **Hook :** Qu'est-ce qui capture l'attention ?
- **Problem :** Quel pain point agite-t-on ?
- **Promise :** Quel résultat promis ?
- **Proof :** Quelles preuves ?
- **Offer :** Comment est structurée l'offre ?
- **CTA :** Comment demande-t-on l'action ?

#### SECTION 4 : USAGE ETHICS
**S'inspirer, pas plagier :**
- Extraire le pattern, pas le wording
- Adapter à ton audience/produit
- Test avant de scaler
- Créditer quand approprié
- Créer ton propre style

### 💡 PRINCIPES CLÉS

1. **Quality over quantity** — Mieux vaut 50 swipes analysés que 500 non lus
2. **Organize by use case** — Facile à retrouver quand tu en as besoin
3. **Regular review** — Swipe file = living document
4. **Test before assuming** — Ce qui marche pour eux peut ne pas marcher pour toi
5. **Build your own** — Au fil du temps, créer tes propres winners

### 🛠️ APPLICATION PRATIQUE

**Swipe File Setup :**
1. Choisir tool (Notion, Airtable, Google Drive)
2. Créer catégories pertinentes
3. Commencer collection (10 items/jour)
4. Analyser chaque item avec framework
5. Review weekly pour patterns

### 📊 EXEMPLE CONCRET

**Swipe File Impact :**
- **Collection :** 200+ Facebook ads analysés
- **Pattern identified :** "How I [result] without [common objection]" = format gagnant
- **Application :** Adapté à produit e-commerce
- **Test :** 3.2% CTR vs 1.8% baseline
- **Scaling :** Devenu angle principal, $500K+ revenue
