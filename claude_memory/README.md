# Mémoire Claude Code — Session ECOM SLAVE

Ce dossier contient toute la mémoire structurée d'une session Claude Code travaillant sur **ECOM SLAVE** (ex-OMNIA), le Creative OS de Chef pour drinknellio.com.

## À faire pour intégrer dans une autre session Claude Code

1. **Lire d'abord `MEMORY.md`** — c'est l'index, il pointe vers tous les autres fichiers
2. Lire ensuite chaque fichier référencé dans l'ordre où ils apparaissent
3. Copier ces fichiers dans le système de mémoire de l'autre session :
   ```bash
   cp claude_memory/*.md ~/.claude/projects/<your-project-slug>/memory/
   ```
