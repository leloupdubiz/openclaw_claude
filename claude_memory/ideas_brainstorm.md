## 1. Plateforme de formation privée (type Skool)
**Date :** 2026-04-05

**Concept :** Plateforme de cours privée où Chef dispense ses formations.

**Mécanique clé :** Cours théorique (texte/marketing) → Examen quiz de validation → Accès débloqué à la vidéo d'application pratique.

**Pourquoi c'est intéressant :**
- Force les élèves à vraiment lire et comprendre la théorie avant de passer à la pratique
- Différenciation vs Skool/Teachable où les gens skip direct aux vidéos sans comprendre le fondement
- Gamification naturelle (progression, déblocage)
- Réutilise la bibliothèque existante (472 docs formations, 62 livres, 113 vidéos)

**Questions ouvertes :**
- Quel sujet en premier ? (EVOLVE ? E-commerce DTC ? AI pour e-com ?)
- Pricing model ? (abonnement mensuel type Skool, one-time, cohort-based ?)
- Self-hosted vs plateforme existante customisée ?
- Intégration avec OMNIA possible ? (nouveau module)
- Communauté intégrée (forum/discord) ou juste cours ?

**Statut :** Idée — à développer

## 2. Remplacer agences & freelances e-com par des agents/SaaS
**Date :** 2026-04-05

**Concept :** Chaque rôle qu'un entrepreneur e-commerce paie à une agence ou un freelance peut être remplacé par un agent IA ou un SaaS autonome. Construire ces outils et les vendre.

**Rôles à remplacer :**
| Rôle humain | Coût mensuel typique | Remplacement agent/SaaS |
|-------------|---------------------|------------------------|
| Media buyer | 1500-5000€ | Agent EVOLVE (campagnes, budgets, kill/scale) |
| Creative strategist | 1000-3000€ | Agent angles/hooks/scripts (déjà dans OMNIA) |
| UGC creator | 500-2000€/vidéo | Studio Clip (kie.ai/HeyGen/Sora) |
| Graphic designer | 500-1500€ | Image Factory (déjà dans OMNIA) |
| Copywriter | 500-2000€ | Script Writer agent |
| Data analyst | 1000-3000€ | Compta/Triple Whale + agent analyse |
| CRO specialist | 1500-4000€ | CRO agent + A/B testing auto |
| Email marketer | 500-2000€ | Klaviyo + agent flows |
| Market researcher | 1000-3000€ | EVOLVE Research pipeline |
| Vidéaste/monteur | 1000-5000€ | Video Cloner + UGC pipeline |

**Pourquoi :** Un opérateur DTC paie 10-30K€/mois en agence/freelances. OMNIA remplace déjà une partie. L'objectif : remplacer 80%+ pour <200€/mois de SaaS.

**Approche :** Dupliquer les meilleurs SaaS du marché en mieux + intégré :
- Foreplay → AdSpy Pro (déjà fait)
- Triple Whale → Compta (déjà fait)
- Skool → Plateforme formation (idée #1)
- Motion/Sovran → Creative analytics
- Atria → Ad Library scraping
- Replo → Landing page builder
- Pencil/AdCreative → Générateur créatif IA

**Avantage compétitif :** Tout intégré dans un seul OS (OMNIA) vs 10 abonnements séparés.

**Statut :** Vision long-terme — OMNIA est déjà le prototype

## 3. Modules OMNIA manquants — combler les trous

**Existant vs manquant :**

| Fonction | Module OMNIA | Statut | Remplace |
|----------|-------------|--------|----------|
| Research marché | EVOLVE | ✅ Done | Market researcher |
| Veille concurrentielle | AdSpy Pro | ✅ Done | Foreplay/Atria |
| Analytics/attribution | Compta (Triple Whale) | ✅ Done | Triple Whale |
| Génération UGC vidéo | Studio Clip | ✅ Done | UGC creators |
| Scripts/hooks/angles | EVOLVE agents | ✅ Done | Copywriter/strategist |
| Knowledge base | Bibliothèque | ✅ Done | Notion/docs internes |
| OKR/tâches | Command Center | ✅ Done | Asana/Monday |
| **Landing page builder** | — | ❌ Manquant | Replo/Unbounce |
| **Email/SMS automation** | — | ❌ Manquant | Klaviyo (agent) |
| **Creative analytics** | — | ❌ Manquant | Motion/Sovran |
| **A/B testing auto** | — | ❌ Manquant | VWO/Optimizely |
| **Landing page CRO** | — | ❌ Manquant | Replo + agent CRO |
| **Post-purchase flows** | — | ❌ Manquant | Klaviyo flows agent |

**Priorité suggérée (impact × faisabilité) :**
1. **Creative Analytics** — Analyser quels créatifs performent, pourquoi, et quoi tester ensuite. Les données viennent déjà de Triple Whale. Faisabilité haute.
2. **Landing Page Builder** — Générer des LP optimisées à partir des angles EVOLVE. Template-based + IA. Impact direct sur CVR.
3. **Email/SMS Automation** — Agent qui génère des flows Klaviyo (welcome, post-purchase, winback) à partir des avatars EVOLVE. Se branche sur l'API Klaviyo.
4. **A/B Testing Auto** — Tester variantes de LP/ads automatiquement avec règles de kill/scale.

**Statut :** À planifier — prioriser après stabilisation des modules existants
