## 5 Frameworks Clés

| # | Framework | Principe | Impact estimé |
|---|-----------|----------|---------------|
| 1 | **Desire-First** | Channel un desire existant, ne vends pas un produit | +20-40% ROAS |
| 2 | **5 Awareness Levels** | Unaware → Problem → Solution → Product → Most Aware | Scalabilité |
| 3 | **Sub-Avatar System** | Core avatar → Sub-avatars (15-30) → Angles spécifiques | Relevance |
| 4 | **3-2-2 Testing** | 3 créatifs × 2 body copies × 2 headlines = 12 ads/test | Testing velocity |
| 5 | **Marksman → Sniper → Scale** | Test 3 angles larges → Deep-dive winner → Scale | Hit rate |

## Architecture Agents (20+ mappés, dans OMNIA)

### RESEARCH LAYER
- Market Researcher — Desires dominants, demand mapping
- Avatar Architect — 3 Core Avatars (Primary Desire, Fear, Awareness, Micro-insight)
- Ad Library Spy — Monitoring concurrents, winning ads, créatives qui tournent
- Research Synthesizer — Connexion insights inter-sources
- Sub-Avatar Specialist — 15+ sub-avatars avec micro-insights
- Review Analyzer — Mining Amazon/Reddit/Trustpilot verbatims
- BrandSearch + TrendTrack — Ad spend réel, scaling patterns, trafic
- Research Doc Generator — Master Research Doc (20 désirs, 20 verbatims, 30 hooks)

### STRATEGY LAYER
- Angle Extractor — 30-90 angles depuis les desires
- Concept Strategist — Planning batches Marksman/Sniper

### CREATION LAYER
- Hook Writer — Hooks 3 secondes
- Script Writer — Scripts UGC/VSL en allemand
- Brief Creator — Briefs créatifs pour créateurs

### EXECUTION LAYER
- Campaign Builder — Setup Meta (3-2-2)
- Champion Scaler — Scale des winners

### ANALYSIS LAYER
- Performance Analyst — Winners/losers, reporting

## Séquence obligatoire des phases
```
PHASE 1 — RESEARCH : Desire mapping → competitors → consolidation
PHASE 2 — STRATEGY : Avatars → Sub-avatars → Angles → Creative roadmap
PHASE 3 — CRÉATION : Hooks → Scripts → Briefs
PHASE 4 — EXÉCUTION : Campagnes Meta → 72h data
PHASE 5 — SCALE : Performance analysis → Scaling plan
```

Ne jamais sauter une phase. Chaque output alimente le suivant.
Marksman d'abord (test 3 angles larges) avant Sniper (deep-dive sur winner).
