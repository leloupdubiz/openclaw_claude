## A. PRINCIPE #1 — DESIRE-FIRST (Schwartz + EVOLVE)
Tu ne vends pas un produit. Tu channels un désir existant.
- Le désir est la force. Le produit est le véhicule.
- Desire Strength = Intensité (1-100) × Scope (1-100)
- "Enter the conversation already taking place in the customer's mind" — Collier

## B. 5 NIVEAUX D'AWARENESS (EVOLVE)
| Niveau | État client | Message |
|--------|-------------|---------|
| Unaware | Ne sait pas qu'il a un problème | Identifier la douleur |
| Problem-Aware | Sait qu'il souffre | Agiter, rendre réel |
| Solution-Aware | Cherche des solutions | Présenter le mécanisme |
| Product-Aware | Connaît le produit | Lever les objections |
| Most Aware | Hésite encore | Urgence + risk removal |
→ TOF Meta : cibler Problem/Solution-Aware

## C. ARCHITECTURE AVATAR → ANGLE
```
1 Core Avatar (1 désir dominant)
├── Sub-Avatar 1 → Angles A1, A2, A3
├── Sub-Avatar 2 → Angles B1, B2, B3
└── Sub-Avatar 3 → Angles C1, C2, C3
```
Objectif : 1-3 core → 15-30 sub-avatars → 30-90 angles
Angle ≠ Format. Angle = raison d'acheter.

## D. TESTING 3-2-2
3 créatifs × 2 body × 2 headlines = 12 ads/batch
- **Marksman** (pas de data) : 3 angles différents, 72h → trouver la direction
- **Sniper** (angle winner trouvé) : 1 angle, 10 hooks × variations, 7-14j
- Plateau ? → Marksman à nouveau

## E. HOOK — RÈGLE DES 3 SECONDES
```
HOOK = Callout audience + Promesse bénéfice + Curiosity gap
```
4 types : Résultat idéal | Clip bizarre | Démo problème | Démo produit
5 leviers émotionnels (Pellegrini) : Encourager rêves | Justifier échecs | Calmer peurs | Confirmer soupçons | Donner un ennemi

## F. STRUCTURE SCRIPT
```
SCROLL → HOOK (0-3s) → PROBLEM (3-8s) → BRIDGE → MÉCANISME → PREUVE → CTA
```

## G. META ADS — SEUILS DÉCISION
| Métrique | Action |
|----------|--------|
| ROAS > 3.0 (72h) | Scale +20% |
| ROAS 2.0-3.0 | Hold, optimiser hooks |
| ROAS < 2.0 (48h) | Analyser, itérer |
| CPA > €60 (48h) | Stop ad set |
| Hook Rate < 25% | Changer hook |
| Hold Rate < 40% (3s) | Revoir les 3 premières secondes |

CBO : €50-75/j min, max 3 ad sets, max 15 ads/ad set, 72h min avant décision.

## H. MATHS E-COMMERCE
- **Shopify Score** = AOV × CVR% → cible > 250
- **LTV** = 1er achat + Upsell checkout + Reorder J+30
- **Loi Pellegrini** : celui qui peut dépenser le plus en acquisition gagne
- Si LTV = 2× CPA → tu surenchéris tous les concurrents
- Marge brute minimum viable : 34%

## I. CRO — HARMONIE AD↔LP
- Ad angle "Cortisol" → LP mentionne Cortisol en premier
- Ad avatar "Manager" → LP montre des managers
- Social proof above fold, prix visible sans scroll, vidéo démo en hero
- Upsell au checkout (3 mois → 6 mois)

## J. SCALING (Pellegrini)
1. Test (€300-1K/j) → Identify winners → Iterate
2. Fix bottlenecks avant d'augmenter budget
3. **Horizontal** > Vertical : dupliquer sur 2-3 ad accounts, partnership ads, whitelisting
4. Creative Coherence Index : B-roll qualité, expressions authentiques, pacing dynamique, preuve intégrée

## K. RETENTION / LTV
Post-purchase emails : J+3 onboarding → J+7 social proof → J+14 science → J+30 upsell → J+45 review
Cibles : repeat rate >30% (60j), orders >1.8 (6 mois), email open >25%, upsell conv >15%

## L. OFFRES ($100M Offers — Hormozi)
- L'offre est la pierre angulaire de tout marketing rentable
- Value Stack : produit + bonus + garantie + scarcity + urgence
- Price = f(valeur perçue), pas f(coût)
- "If you can't explain your offer in 16 words, you don't understand it"

## M. COPYWRITING (Sugarman + Schwartz + Hopkins)
- "The sole purpose of the first sentence is to get you to read the second"
- 31 triggers psychologiques (Sugarman)
- Publicité = vendeur imprimé (Hopkins, 1923)
- Leads : la partie la plus critique de tout message de vente (Masterson)
