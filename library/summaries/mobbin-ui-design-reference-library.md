# Mobbin — Bibliothèque de Référence UI/UX
> mobbin.com | Librairie design · Screenshots · Flows · Pricing · PDP · Landing pages

## 🎯 Concept Central
Mobbin est la plus grande bibliothèque de screenshots UI provenant de vraies applications mobiles et web. Chaque écran est catégorisé par type (onboarding, pricing, checkout, landing page) et par industrie. Indispensable pour benchmarker son UX et s'inspirer des meilleures pratiques sans partir de zéro.

## 📌 Points Clés
- **+100,000 screenshots** d'apps réelles (iOS, Android, Web) — Apple, Shopify, Netflix, Calm, Headspace...
- **Filtres puissants** : par flow (onboarding, checkout, paywall), par composant (modal, CTA, form), par industrie
- **Flows complets** : voir le parcours entier d'une app (sign-up → first use → upsell → churn prevention)
- **Pricing pages** : référence pour structurer ses propres pages tarifaires SaaS
- **Landing pages** : 1000+ exemples catégorisés — hero, social proof, FAQ, CTA placement
- **PDP e-commerce** : analyser comment les meilleurs convertissent (Gymshark, AG1, OLIPOP, etc.)
- Mise à jour constante — nouvelles apps ajoutées chaque semaine
