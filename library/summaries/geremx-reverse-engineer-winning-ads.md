## 📋 Le Pipeline en 5 Étapes

### Étape 1 — Nettoyage des Sous-Titres (Vmake)
```
→ Passer la créative concurrente sur Vmake
→ Nettoyage complet des sous-titres existants
→ Résultat : vidéo propre sans texte concurrent
```

### Étape 2 — Extraction Audio + Analyse Structure (ElevenLabs + Claude)
```
→ Extraire l'audio sur ElevenLabs
→ Soumettre à Claude : "Analyse la structure gagnante de ce script"
   → Identifier : hook, problème, mécanisme, preuve, CTA
   → Déconstruire le rythme, les transitions, les longueurs de phrases
→ Comprendre POURQUOI ça convertit (pas juste quoi)
```

### Étape 3 — Réécriture du Script (Claude Project)
```
→ Dans un projet Claude avec :
   - Documents fondamentaux (SOUL.md, MEMORY.md, brand guidelines)
   - Recherche marketing (desire map, avatars)
   - Ressources copywriting (Breakthrough Advertising, Eugene Schwartz etc.)

→ Réécrire le script adapté à TON produit / TA brand
→ Conserver la STRUCTURE gagnante, changer le CONTENU
```

### Étape 4 — Extraction des Éléments Visuels (CapCut)
```
→ Importer la créative concurrente dans CapCut
→ "Division par scènes" → découpe automatique par cut
→ "Exporter les clips sélectionnés" → récupère :
   • Hooks visuels (les 3 premières secondes)
   • B-rolls génériques réutilisables
   • CTA (call-to-action)
   • B-rolls produit intéressants
→ Ces éléments alimentent ta bibliothèque de B-rolls
```

### Étape 5 — Remontage de ta Créative
```
→ Nouvelle voix off avec ton script (ElevenLabs ou UGC)
→ B-rolls existants récupérés (step 4)
→ Tes propres B-rolls perso (UGC, Freepik, tournage)
→ Résultat : créative avec la STRUCTURE gagnante concurrente
             + TON produit + TA voix + TES visuels
```

## 🏗️ La Bibliothèque de B-Rolls (Levier Long Terme)
```
Règle : 1h/jour minimum pour :
→ Trier et classer tes B-rolls récupérés
→ Alimenter la bibliothèque avec tes propres tournages
→ Tags : hook / broll-produit / testimonial / CTA / transition / lifestyle

Effet cumulé :
→ Semaine 1 : 30 min de montage par créative
→ Mois 3 : 10 min de montage (bibliothèque riche)
→ Mois 6 : montage = jeu, script = travail principal
```
