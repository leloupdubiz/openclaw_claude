# Claude Code + Nano Banana 2 — Product Shots DTC Automatisés
> @mikefutia (Kitsch & MAELYS) | 135 bk | Claude Code · Nano Banana 2 · Texte→JSON→API · DTC

## 🎯 Concept Central
Pipeline automatisé pour générer des product shots e-commerce de qualité premium : Claude Code transforme un brief texte en JSON structuré, qui est envoyé à l'API Nano Banana 2 pour générer des visuels produit photorealistes. Utilisé en production chez Kitsch et MAELYS — marques DTC multi-millions.

## 📌 Points Clés
- **Workflow** : Brief textuel → Claude Code → JSON prompt structuré → Nano Banana 2 API → Image produit
- **Nano Banana 2** : Modèle image spécialisé product photography — meilleur que Midjourney pour le e-commerce
- **Claude Code** rédige le JSON de prompt selon les specs NB2 (angle, lighting, background, mood, product placement)
- Coût estimé : ~$0.05-0.10 par image vs $200-500 pour un shoot studio
- Batch generation : 50 variantes d'un même produit en 10 minutes
- Style cohérence : JSON template réutilisable pour maintenir l'identité visuelle
- Output formats : PNG 4K, WebP optimisé pour Shopify, carré pour Meta Ads

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
- **Nellio UltraCalm** : Générer 20+ product shots avec décors variés (bureau zen, chambre, nature) pour Meta Ads statiques DE
- **Nellio Studio** : Intégrer Nano Banana 2 comme provider image — bouton "Generate Product Shot" avec brief Claude
- **EVOLVE Phase 3** : Brief Creator → JSON → NB2 → images directement uploadées dans Meta Ads Manager
- **OMNIA SaaS** : Module "Product Studio" pour les clients DTC — automatiser leurs shoots à $0.10/image
