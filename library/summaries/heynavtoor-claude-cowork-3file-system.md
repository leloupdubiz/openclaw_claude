# Claude Cowork 10× ChatGPT — Système 3 Fichiers
> @heynavtoor | 2083 bk | Claude · 10× ChatGPT · about-me.md · brand-voice.md · working-style.md

## 🎯 Concept Central
Remplacer ChatGPT par Claude en utilisant un système de 3 fichiers de contexte persistants. Ces 3 fichiers transforment Claude en un collaborateur qui connaît ton business, ton style, et tes préférences — résultant en des outputs 10× plus pertinents que ChatGPT avec des prompts génériques.

## 📌 Points Clés
- **Fichier 1 — about-me.md** : Qui tu es, ton business, ton marché, tes produits, tes KPIs, tes objectifs. Claude comprend le CONTEXTE.
- **Fichier 2 — brand-voice.md** : Ton ton de marque, tes exemples de copies qui marchent, tes mots interdits, ta cible. Claude parle comme ta marque.
- **Fichier 3 — working-style.md** : Tes préférences de format, tes raccourcis, comment tu prends des décisions, tes heures de travail. Claude travaille comme TOI.
- **Usage** : Attacher ces 3 fichiers au début de chaque conversation importante (ou les intégrer dans CLAUDE.md)
- **Mise à jour** : Réviser les 3 fichiers toutes les 2 semaines ou après un pivot
- **Résultat** : 0 temps perdu à ré-expliquer le contexte · Output directement utilisable · Moins d'itérations
- **10× ChatGPT** : ChatGPT oublie tout entre les sessions — ce système maintient la continuité

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
- **Notre CLAUDE.md** est déjà notre "about-me.md" — ajouter un `brand-voice.md` avec exemples de copies DE qui ont performé
- **working-style.md** = formaliser les règles de AGENTS.md §2 (formats de réponse) pour Claude
- **Nellio Studio** : Intégrer les 3 fichiers dans le system prompt du backend → scripts générés toujours cohérents avec la voix Nellio
