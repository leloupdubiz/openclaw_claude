# Perplexica — Clone Local Open-Source de Perplexity AI
> Source : https://x.com/ihtesham2005/status/2028723408670994600 | Auteur : @ihtesham2005 | Date : Mars 2026

## 🎯 Concept Central
Perplexica est un moteur de recherche AI auto-hébergé, alternative gratuite et open-source à Perplexity.ai. Il combine la recherche web en temps réel avec des LLMs locaux (Ollama/LM Studio) pour fournir des réponses sourcées avec citations, sans abonnement ni fuite de données.

## 📌 Points Clés
- **GitHub** : `ItzCrazyKns/Perplexica` — Docker-ready, auto-hébergeable
- Compatible avec Ollama (Llama 3, Qwen, Mistral) et APIs OpenAI/Anthropic
- Recherche web réelle avec SearxNG comme backend search
- Réponses avec citations sources = fiabilité vérifiable
- Interface similaire à Perplexity : questions en langage naturel
- 2 modes : Docker (recommandé) ou installation manuelle Node.js
- Peut être combiné avec des modèles locaux pour zéro coût de tokens
