# AI Cinematic Video Prompts — Penser comme un Réalisateur
> @Parul_Gautam7 | 285 bk · 40K vues | Shot language · JSON template · Grille 3×3 · Timeline prompts

## 🎯 Concept Central
Framework pour écrire des prompts vidéo IA qui produisent des résultats cinématographiques — en pensant comme un réalisateur, pas comme un utilisateur lambda. La clé : utiliser le langage technique du cinéma (shot type, camera movement, lighting) dans une structure JSON réutilisable.

## 📌 Points Clés
- **Shot language** : Wide shot, close-up, POV, dolly-in, rack focus — chaque terme change radicalement le résultat
- **JSON template** : `{"shot_type": "...", "camera_movement": "...", "lighting": "...", "mood": "...", "subject": "...", "action": "...", "duration": "..."}`
- **Grille 3×3** : 3 types de shots × 3 moods = 9 variations cohérentes pour un même produit
- **Anchor phrase** : Phrase clé qui reste constante dans toutes les variations pour la cohérence de marque
- **Timeline prompts** : Séquencer les prompts pour raconter une histoire (Problem → Tension → Resolution → CTA)
- Outils recommandés : Sora, Runway, Kling, Higgsfield — chacun avec ses forces
- B-rolls séparés des A-rolls : générer le contexte et le produit séparément, merger en post

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
- **Nellio UltraCalm UGC** : Template JSON pour vidéos stress/sommeil — close-up mains avec verre, dolly-out sur visage détendu, golden hour
- **Higgsfield integration** : Nos B-rolls Nellio générés avec ce framework JSON → plus cohérents, moins de rejets
- **EVOLVE Phase 3** : Script Writer intègre le framework shot-language dans ses briefs vidéo → output créateur/IA plus précis
- **OMNIA** : Module "Video Director AI" — le client décrit le résultat voulu, OMNIA génère le JSON de prompt optimal

## ⚡ Citation / Insight Clé
> "Le prompt vidéo IA parfait n'est pas une description — c'est un storyboard textuel. Shot type + movement + light + mood + duration."
