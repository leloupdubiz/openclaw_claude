# Héberger OpenClaw sur un Vieux Téléphone Android
> Source : https://x.com/itspaulai/status/2028213201519640823 | Auteur : @itspaulai | Date : Mars 2026

## 🎯 Concept Central
Un vieux smartphone Android peut devenir un serveur OpenClaw/Claude Code 24/7 à coût zéro. Anthropic a lancé "Remote Control" pour Claude Code permettant d'émettre des commandes depuis mobile. Cette approche transforme un appareil obsolète en nœud AI permanent sans coût infrastructure.

## 📌 Points Clés
- Vieux Android (≥Android 10) = serveur AI local permanent
- Claude Code Remote Control : commandes depuis iOS/Android vers l'agent desktop
- Termux + SSH permet de faire tourner des scripts AI sur Android
- Coût : $0 hardware (réutilisation) + électricité minimale
- OpenClaw Gateway peut se connecter à un nœud Android via pairing
- Accessible n'importe où : commande l'agent depuis son téléphone principal
- Anthropic Claude Max ($100-$200/mois) requis pour Remote Control officiel

## 💡 Insights Actionnables
1. Installer Termux + SSH sur un vieux Android pour en faire un serveur 24/7
2. Configurer OpenClaw Gateway pour se connecter à distance à ce nœud
3. Utiliser Remote Control de Claude Code pour superviser des tâches depuis mobile
4. Brancher le téléphone en permanence = serveur free tier sans limite de temps
