# osinttools.io — Annuaire Complet des Outils OSINT
> @Cyber_Sudo | 231 bookmarks | OSINT · Username · Domain · Image · Social

## 🎯 Concept Central
osinttools.io est un annuaire centralisé de plus de 500 outils OSINT (Open Source Intelligence) classés par catégorie : recherche de personnes, analyse de domaines, reconnaissance d'images, surveillance des réseaux sociaux, et plus. Ressource de référence pour tout audit de présence en ligne ou recherche de compétiteurs.

## 📌 Points Clés
- **Username lookup** : Sherlock, WhatsMyName — retrouver tous les comptes liés à un pseudo
- **Domain intel** : WHOIS, Shodan, VirusTotal, SecurityTrails — analyser l'infrastructure d'un domaine
- **Image reverse** : TinEye, Google Images, Yandex Images — tracer l'origine d'une image
- **Social monitoring** : Social Search, Followerwonk, Tweetbeaver (données profils publics)
- **Phone/Email** : Hunter.io, Phonebook.cz — trouver des contacts professionnels
- **Geolocation** : Geospy, Suncalc — analyser des photos geolocalisées
- Mise à jour régulière de l'annuaire — filtres par catégorie, gratuité, légalité

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
- **Competitor intel** : Analyser les domaines concurrents (traffic, infra, email providers) pour benchmarker drinknellio.com
