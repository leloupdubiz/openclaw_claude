## 🎯 Concept Central
Scrapling est une librairie Python de scraping nouvelle génération qui combine une vitesse 774× supérieure à BeautifulSoup avec la capacité de bypasser Cloudflare et les protections anti-bot modernes. Idéal pour miner des reviews, des prix, des ads compétiteurs sans être bloqué.

## 📌 Points Clés
- **774× plus rapide** que BeautifulSoup sur des pages identiques (benchmark officiel)
- **Bypass Cloudflare** : utilise des techniques de fingerprinting browser pour passer inaperçu
- **Async natif** : `asyncio` pour scraper des centaines de pages en parallèle
- **API simple** : `from scrapling import Scraper` — interface similaire à BeautifulSoup mais plus puissante
- Auto-adaptation : si le HTML change, Scrapling re-map automatiquement les sélecteurs
- Compatible avec proxies rotatifs pour des opérations à grande échelle
- Installation : `pip install scrapling` — zéro dépendance binaire externe
