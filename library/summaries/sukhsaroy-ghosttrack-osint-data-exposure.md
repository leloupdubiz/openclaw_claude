# GhostTrack — Phone/IP/Username → Localisation
> @sukh_saroy | 1235 bk | GhostTrack · OSINT · 5 protections · 6K GitHub stars

## 🎯 Concept Central
GhostTrack est un outil OSINT open source (6K+ étoiles GitHub) qui permet de corréler un numéro de téléphone, une adresse IP, ou un nom d'utilisateur pour obtenir des informations de localisation et de profil. Outil à double tranchant : utilisé pour la recherche légitime ou pour comprendre son propre niveau d'exposition.

## 📌 Points Clés
- **Inputs acceptés** : Numéro de téléphone · Adresse IP · Username / pseudo · Email
- **Outputs possibles** : Localisation approximative · Profils réseaux sociaux liés · Historique public · Opérateur télécom
- **6K GitHub stars** = outil bien maintenu et largement utilisé par la communauté OSINT
- **5 Protections recommandées** :
  1. VPN permanent (NordVPN, Mullvad) — masquer l'IP réelle
  2. Pseudonymes différents par plateforme — éviter la corrélation
  3. Numéro de téléphone virtuel (Hushed, MySudo) pour les inscriptions
  4. Email jetable pour les services non critiques
  5. Vérification régulière de sa propre exposition (scan GhostTrack sur soi-même)
- Légalité : OSINT sur données publiques = légal dans la majorité des pays si usage non malveillant

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
- **Audit exposition** : Scanner les comptes liés à drinknellio.com et nos profils pros — identifier ce qui est public
- **Concurrents** : Profiler les fondateurs de marques concurrentes DE pour comprendre leur background et réseaux
- **OMNIA security** : Proposer un audit OSINT dans l'onboarding client — "voici ce qu'un concurrent peut voir de vous"
- **Hygiène numérique** : Appliquer les 5 protections pour nos comptes business (Meta, HeyGen, Anthropic)
