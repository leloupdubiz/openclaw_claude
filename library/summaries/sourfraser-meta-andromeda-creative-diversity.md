## 🎯 Concept Central

**La mise à jour Andromeda de Meta a changé la logique de diffusion des ads.** Avant : "À qui montrer cette pub ?" → Maintenant : "Quelle version de cette pub montrer à cette personne ?" Conséquence directe : changer uniquement le hook ne suffit plus. Il faut une vraie diversité créative — 4 leviers distincts, pas 1.

> *"Creative is your targeting. The algorithm finally caught up to that reality."*

---

## 📋 Les 4 Leviers de la Diversité Créative

### Levier 1 — Persona (même produit, acheteurs différents)

```
Exemple : Humantra (complément hydratation) 
→ Persona A : Athlètes / coureurs marathon (peak performance)
→ Persona B : Office workers qui oublient de boire (maux de tête l'après-midi)

Même produit. Acheteurs totalement différents. Ads différentes.
```

**Application Nellio DE :**

| Persona | Angle | Trigger |
|---------|-------|---------|
| Markus (cadre 35ans) | Cortisol professionnel | Réveil 3h nuit avant réunion importante |
| Sonja (mère 38ans) | Sommeil fragmenté | Enfants + job + couple = épuisement |
| Julia (étudiante 22ans) | Stress examens | Cerveau qui tourne la nuit |

### Levier 2 — Messaging (raison différente d'acheter)

```
Même persona, angle différent :
→ Athlètes message A : performance pendant le marathon
→ Athlètes message B : récupération post-effort

Same audience. Completely different value proposition.
```

### Levier 3 — Hook (attention dans les 3 premières secondes)

```
Types de hooks :
→ Visual hook (image choc, geste inattendu)
→ Audible hook (son/mot intrigant en opening)
→ Copy hook (texte on-screen : question, claim, chiffre)
→ Bold claim ("J'ai dormi 8h d'affilée pour la première fois en 3 ans")
→ Demo visuelle (produit en action)

"Your job is pulling people away from their feed filled with friends, family, and influencers."
```

### Levier 4 — Format (wrapper de l'ad)

```
UGC testimonial · Vidéo fondateur · Statique · Carrousel · 
Demo produit · Podcast / interview expert · Voiceover VSL

Le format change la perception même si le contenu est identique.
```

## 📋 Process Fraggle en 5 Étapes (Méthode Concrète)

```
Étape 1 — Lock Persona + Messaging
→ Persona : Athlètes
→ Messaging : Humantra = peak performance marathon

Étape 2 — Choisir le Format (basé sur data historique)
→ Podcast format fonctionne bien → 2 options :
  - Expert interview : marathonien interviewé sur l'hydratation
  - Peer conversation : 2 coureurs s'échangent des tips

Étape 3 — Tester et Itérer
→ Si podcast ne marche pas → changer le format :
  UGC testimonial : quelqu'un en run documentant sa préparation

Étape 4 — Layering les Hooks
→ Podcast : "How this simple hack altered my marathon running forever"
→ UGC : quelqu'un qui verse le sachet sur son visage (pattern interrupt visuel)

Étape 5 — Scaler sur tout le compte
→ Nouveau persona + nouveau messaging + nouveau format = vraie diversité
→ Pas "même ad, hook différent"
```

## 🚨 Ce qui Ne Fonctionne Plus (Andromeda)

```
❌ Même script · même créateur · même footage → hooks différents
❌ Même ad avec des petits tweaks de caption ou d'image
❌ Volume d'ads sans vraie diversité

Meta détecte que c'est "essentially the same ad" → saturation → performance en chute

"Creative volume is NOT the answer. Intentional variety is."
```

---

## ✅ Ce qui Fonctionne Post-Andromeda

```
✅ Trouver un angle winner → aller PLUS PROFOND sur cet angle (pas le cloner)
✅ Même script → créateur différent + lieu différent + format différent
✅ 5-6 créatifs vraiment différents plutôt que 20 variations du même
✅ Laisser le système choisir la bonne version pour la bonne personne
```
