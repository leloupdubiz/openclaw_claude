# Beginner Roadmap — AI Agents 2026
> @mkbijaksana (ex-Google/Microsoft) | 2292 bk · 383K vues | Python · LangGraph · CrewAI · Multi-agents · Deploy

## 🎯 Concept Central
Roadmap complète en 6 mois pour passer de débutant à développeur d'agents IA multi-agents déployés en production. Créé par un ex-Google/Microsoft, entièrement accessible gratuitement. La séquence est non-négociable : Python d'abord, frameworks ensuite, multi-agents en dernier.

## 📌 Points Clés
- **Mois 1 — Python Foundations** : Variables, fonctions, APIs, async/await. Pas de raccourci possible.
- **Mois 2 — LLM Basics** : OpenAI/Anthropic API, prompting, function calling, structured outputs.
- **Mois 3 — LangChain / LangGraph** : Chains, tools, memory, state machines pour agents complexes.
- **Mois 4 — CrewAI** : Framework multi-agents avec rôles, délégation, collaboration entre agents.
- **Mois 5 — Multi-agent Architecture** : Orchestrateur + agents spécialisés, handoffs, error handling.
- **Mois 6 — Deploy** : FastAPI, Docker, Railway/Render, monitoring, coûts en production.
- Ressources gratuites : Python.org, DeepLearning.AI courses, docs LangChain, GitHub examples.

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
- **EVOLVE Agents** : Suivre ce roadmap pour construire les 6 agents EVOLVE (Desire Researcher → Champion Scaler) avec CrewAI
- **OMNIA SaaS** : Architecture multi-agents = backbone technique de la plateforme — LangGraph pour les workflows complexes
- **Nellio Studio v2** : Migrer vers LangGraph pour orchestrer les pipelines créatifs (brief → hook → script → image prompt)
- **Formation interne** : Roadmap de référence pour monter en compétence sur les agents avant de tout externaliser

## ⚡ Citation / Insight Clé
> "Don't start with CrewAI. Start with Python. Every shortcut at month 1 becomes a bug at month 5."
