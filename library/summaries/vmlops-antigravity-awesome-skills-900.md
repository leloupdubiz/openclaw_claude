> @_vmlops | 1635 bk | 900+ skills · GitHub · Claude · Copilot · Gemini · Cursor

## 🎯 Concept Central
`antigravity-awesome-skills` est le GitHub le plus complet pour les skills/prompts IA — plus de 900 skills organisées pour Claude, GitHub Copilot, Gemini, et Cursor. C'est le "awesome list" de référence pour l'écosystème des AI coding agents et assistants.

## 📌 Points Clés
- **900+ skills** organisées par catégorie : coding, design, marketing, productivity, data, security
- **Multi-plateforme** : Claude (OpenClaw), Copilot (VS Code), Gemini, Cursor — skills adaptées à chaque plateforme
- **Format standardisé** : Chaque skill = description + use case + prompt template + exemple
- **Contributions communautaires** : Des centaines de contributeurs — qualité variable mais densité exceptionnelle
- **Catégories phares** : Frontend coding · API design · SEO · Ad copywriting · Email marketing · Data analysis
- **Antigravity** : Le project umbrella qui regroupe aussi des tools complémentaires (voir @viktoroddy)
- Installation via ClawHub : `clawhub install antigravity-[skill-name]`

## 💡 Applications pour drinknellio.com / OMNIA / EVOLVE
