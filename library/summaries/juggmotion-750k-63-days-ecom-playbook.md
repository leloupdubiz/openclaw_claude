## 📋 Section 3 — Éviter Holds & Bans

**Chargebacks :**
```
→ Garder < 0.3% (1% = Shopify te ban + prend ton argent)
→ Toujours avoir 1 backup high-risk processor prêt
→ Remboursez les clients en colère OU gift card → ne pas laisser les disputes s'accumuler
```

**Payment setup :**
```
Standard : Shopify Payments
À grande échelle : headless checkout → lie multiple processors + subscriptions
```

**Stock / Opérations :**
```
→ Bookkeeper ou le faire soi-même
→ Forecasting stock rigoureux (lead times, reorders)
```
