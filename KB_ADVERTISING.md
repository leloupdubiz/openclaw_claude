## 🧠 PHILOSOPHIE FONDAMENTALE

### La créative fait le ciblage (Pelegrini + EVOLVE)
> "La créative décide qui voit ta pub, pas les audiences."
Meta lit le message → comprend à qui le montrer → trouve les audiences qui correspondent.
- Parle de stress → Meta trouve les stressés
- Montre une femme 35 ans dans une cuisine → Meta attire des femmes 35 ans
- Utilise le mot "Cortisol" → Meta trouve les health-conscious

### Desire-First, pas Product-First (EVOLVE)
On ne vend pas Nellio UltraCalm. On canalise un désir existant :
**"Wieder ich selbst sein — erholt, klar, gelassen"**
Le produit est le véhicule. Le désir est la force motrice.

## 🎯 FRAMEWORK DE TESTING META

### Structure CBO (Pelegrini 2026)
```
CBO Testing
├── Budget : €65/jour
├── Adsets max : 3 (1 par concept)
├── Ads par adset : max 15
├── Durée minimum : 72h avant décision
└── Séparation stricte des concepts :
    Concept 1 (Founder) → assets founder seulement
    Concept 2 (UGC) → assets UGC seulement
    Concept 3 (Statique) → statiques seulement
```

### Règle 3-2-2 (EVOLVE)
```
3 créatifs (angles) × 2 body copies × 2 headlines = 12 ads par batch
Marksman : 3 angles larges (72h) → identifier winner
Sniper : deep-dive winner → 10 hooks × variations
Scale : ROAS winner × budget + accounts
```

### Seuils de décision Nellio (EVOLVE CRO + Pelegrini)
| Métrique | Action |
|----------|--------|
| ROAS > 3.0 sur 72h | ✅ Scale +20% budget |
| ROAS 2.0-3.0 | 🔄 Maintenir, optimiser hooks |
| ROAS < 2.0 48h | ⚠️ Analyser, itérer |
| CPA > €60 48h | 🛑 Stop adset |
| Hook Rate < 25% | 🔄 Changer le hook |
| Hold Rate < 40% à 3s | 🔄 Réviser les 3 premières secondes |

## 📊 FRAMEWORK ANDROMEDA 2026

### Ce qui a changé
Meta Andromeda = diversité créative obligatoire.
**"Pas plus d'ads. Plus d'ads DIFFÉRENTES."**

### Entity ID — ce que Meta pénalise
| Identique → MÊME Entity ID | Différent → NOUVEL Entity ID |
|-----------------------------|------------------------------|
| Même personne + autre texte | Format différent (vidéo → statique) |
| Même décor + autre lumière | Persona différent (âge, genre, style) |
| Même produit + même setup | Setting différent (bureau → cuisine) |
| Même créateur + autre script | Hook différent (storytime → démo) |

### 3 métriques Andromeda
1. **Spend** → signal de confiance Meta. Là où ça spend = potentiel de scaling
2. **Frequency** → ~1.1 = TOF / >2 = BOF saturé
3. **CPR** → à contextualiser (pas valeur absolue) : haut TOF = normal, bas BOF = normal

## 🚀 FRAMEWORK SCALING

### Séquence obligatoire (Pelegrini)
```
0-5K/day :
  → Tester → identifier winners → itérer
  → Budget : 300 → 500 → 750 → 1000 → 5K

5K-100K/day :
  1. FIXER bottlenecks (avant d'augmenter le budget)
  2. FAIRE MIEUX — qualité créatives (Indice Cohérence)
  3. FAIRE PLUS — volume + horizontal scaling
```

### Horizontal Scaling (unlock du vrai volume)
- Duplication sur 2-3 ad accounts (CPM différent = nouvelles audiences)
- Partnership ads (pages 3rd party : expert, magazine, avatar)
- Whitelisting influenceurs DE (wellness, naturopath, coach)

### Indice de Cohérence (Pelegrini)
Même script → ROAS différent selon :
- Qualité du B-roll (émotionnel > générique)
- Expressions faciales du créateur
- Rythme du montage (dynamique vs statique)
- Intégration de la preuve visuelle (avant/après, stats à l'écran)
- Micro-moments réalistes (boire, verser, réaction)

→ **Améliorer l'indice de cohérence = augmenter le ROAS sans changer le budget**

### Objectif CONDOR Nellio
Un CONDOR = une ad qui spend 500K€+ sur sa durée de vie.
Pour le créer :
- Hook + avatar + niveau d'awareness parfaitement alignés
- Parcours émotionnel complet (5 leviers Pelegrini)
- Indice de cohérence maximum (B-roll natürlich, expressions vraies)
- CTA au bon moment (après le pic émotionnel, pas avant)

## 🏆 TOP INSIGHTS RESEARCH DE (125 verbatims collectés)

### Primary Desire Nellio (score 93/125)
**"Endlich wieder durchschlafen und morgens erholt aufwachen"**
→ Pas "réduire le stress" (trop abstrait)
→ Pas "moins de cortisol" (trop technique)
→ L'envie concrète : se réveiller reposé, récupéré, soi-même

### Top 5 expressions clients à réutiliser EXACTEMENT
1. **"Kein Kopfkino mehr"** → angle Schlaf (stop the rumination loop)
2. **"Endlich wieder ich selbst"** → angle Identité
3. **"Ohne mich zu benebeln"** → objection traitée dans l'ad
4. **"10 Minuten und die Anspannung verschwindet"** → hook temporalité
5. **"Seit ich Nellio nehme, schlafe ich endlich durch"** → verbatim testimoanial parfait

### Gaps concurrents identifiés
1. **Aucun concurrent n'adresse l'angle "Abend-Ritual"** → Nellio peut le posséder
2. **"Pourquoi Melatonin ne suffit pas"** → angle éducatif non exploité
3. **Format poudre stick-pack** = seul Nellio dans ce format (tous les autres = capsules)
4. **Arôme agréable** = pain point universel sur les concurrents (goût "medizinisch")

## 📋 COMPLIANCE META ADS DE

### Claims autorisés ✅
- "Kann dazu beitragen, Stress zu reduzieren" (conditionnel)
- "In Studien untersucht" (référence vague)
- "Natürliche Inhaltsstoffe"
- "4,8 Sterne / 20.000 Bewertungen"

### Claims interdits ❌
- "Heilt", "behandelt", "kuriert"
- Claims thérapeutiques directs
- Avant/après médicaux
- Stats sans source vérifiable

### Format légal body copy
```
[Désir/Douleur] → [Solution naturelle] → [Comment ça marche] → [Preuve sociale] → [CTA + garantie]
```
Toujours finir avec : "45 Tage Geld-zurück-Garantie — ohne Risiko testen."
