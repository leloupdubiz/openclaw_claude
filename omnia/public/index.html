🔮 kie.ai — Vue d'ensemble

Plateforme unifiée donnant accès à +100 modèles IA (image, vidéo, audio) via un seul endpoint. Rétention fichiers : 14 jours. Max 20 req/10s.

InfoValeur
Base URLhttps://api.kie.ai
AuthAuthorization: Bearer {API_KEY}
Créer taskPOST /api/v1/jobs/createTask
Statut taskGET /api/v1/jobs/recordInfo?taskId={id}
CréditsGET /api/v1/account/credits
Rate limit20 req / 10 secondes
Rétention14 jours puis expiration

Modèle Type Paramètres clés Coût Durée
sora-2-text-to-video T2V prompt, aspect_ratio (portrait/landscape), n_frames (10/15), remove_watermark ~40 cr/clip ~10s ou ~15s
sora-2-pro-text-to-video T2V ⭐ + size (standard/high) — audio + lip-sync garanti ~100 cr/clip ~10s ou ~15s
sora-2-image-to-video I2V prompt, image_urls (array), aspect_ratio, n_frames — produit fidèle ~40 cr/clip ~10s ou ~15s
sora-2-pro-image-to-video I2V ⭐ + size (high) — HD + audio + produit fidèle ~100 cr/clip ~10s ou ~15s

📝 Payload exact — Sora 2 T2V
⚠️ Le prompt doit inclure le script DE comme dialogue — Sora 2 lit le texte cité pour générer la voix nativement.
{
  "model": "sora-2-text-to-video",
  "input": {
    "prompt": "iPhone 15 Pro selfie, woman in kitchen, holds Nellio
               stick-pack, she says: \"Ich hatte ständig Kopfkino...\"",
    "aspect_ratio": "portrait | landscape",
    "n_frames": "10 | 15",        // string obligatoire
    "remove_watermark": true,
    "size": "standard | high"     // Pro seulement
  }
}
📝 Payload exact — Sora 2 I2V
{
  "model": "sora-2-image-to-video",
  "input": {
    "prompt": "string",
    "image_urls": ["https://..."],  // tableau — URL publique JPEG/PNG/WEBP max 10MB
    "aspect_ratio": "portrait | landscape",
    "n_frames": "10 | 15",
    "remove_watermark": true
  }
}
📊 Statut de task — Réponse
GET /api/v1/jobs/recordInfo?taskId={id}

{
  "data": {
    "state": "generating | success | failed",
    "progress": 0-100,           // % avancement
    "resultJson": "{\"resultUrls\":[\"https://...\"]}",
    "failCode": null,
    "failMsg": null,
    "createTime": 1234567890000
  }
}
États : generating → en cours · success → URL dans resultJson · failed → voir failMsg
⚠️ Erreurs fréquentes
taskId: null → Payload invalide (champ inconnu ou image_urls vide pour I2V)
401 Unauthorized → API key incorrecte ou manquante
n_frames en int → Doit être string : "10" pas 10
resolution / duration → Paramètres invalides ignorés silencieusement par l'API
