## 1. VUE D'ENSEMBLE

### Architecture EVOLVE

La méthodologie EVOLVE est un **système de création d'ads DTC** basé sur 4 piliers :

```
┌─────────────────────────────────────────────────────────────┐
│                    THE EVOLVE FRAMEWORK                      │
├─────────────┬─────────────┬─────────────┬───────────────────┤
│  PSYCHOLOGY │  RESEARCH   │  CREATION   │    EXECUTION      │
│   (Module   │   (Module   │   (Module   │     (Modules      │
│    2-3)     │    3-5)     │    6-7)     │      8-14)        │
├─────────────┴─────────────┴─────────────┴───────────────────┤
│                    CORE PRINCIPLES                           │
│  • Desire-first (pas product-first)                         │
│  • Avatar-based marketing (sub-avatars)                     │
│  • Angle-driven creativity (pas format-driven)              │
│  • 3-2-2 Testing Method                                     │
│  • Marksman → Sniper → Scale workflow                       │
└─────────────────────────────────────────────────────────────┘
```

### Les 5 Concepts Clés à Maîtriser

| Concept | Définition EVOLVE | Importance |
|---------|------------------|------------|
| **Desire** | Force qui fait fonctionner la pub (pas créable, seulement channelable) | ⭐⭐⭐⭐⭐ |
| **Awareness Levels** | 5 niveaux : Unaware → Problem → Solution → Product → Most Aware | ⭐⭐⭐⭐⭐ |
| **Avatar/Sub-avatar** | Core avatar (desire) → Sub-avatars (behaviors/pain points) | ⭐⭐⭐⭐⭐ |
| **Concept vs Angle** | Concept = ce qu'on teste / Angle = raison d'achat du client | ⭐⭐⭐⭐⭐ |
| **3-2-2 Method** | 3 créatifs × 2 body copies × 2 headlines par test | ⭐⭐⭐⭐⭐ |

### MODULE 5 : 📋 HOW TO PLAN ADS (Preparation)

**Concepts Clés :**
1. **Ad Definitions** - Concept vs Angle vs Variation vs Format
2. **3 Testing Methods :**
   - **Marksman** - 3 angles différents (1 variation chacun)
   - **Sniper** - 1 angle en profondeur (3 variations)
   - **Shotgun** - Test massif (à éviter)
3. **Creative Roadmap**
4. **Evolve Growth Guide**

**Framework : Testing Strategy Decision Tree**
```
PAS DE DATA ? → MARKSMAN (trouver direction)
ANGLE GAGNANT ? → SNIPER (maximiser)
PLATEAU ? → MARKSMAN (nouvelle direction)
```

**Framework : 3-2-2 Structure**
```
BATCH #X: [Concept]
├── CREATIVE 1, 2, 3 (variations)
├── BODY COPY 1, 2
└── HEADLINE 1, 2
```

**Agents Nécessaires :**
- 📋 **Concept Strategist** - Définir concepts
- 🎯 **Angle Definer** - Clarifier angles
- 📝 **Variation Writer** - Créer 3 variations
- 📊 **Testing Planner** - Planifier batches
- 📖 **Growth Guide Manager** - Documenter
