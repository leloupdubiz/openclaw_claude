## 🎯 Concept Central
Guide de setup complet et définitif — de zéro à ClawdBot pleinement opérationnel, avec focus sur les configurations qui font vraiment la différence.

## 📌 Points Clés

### Pourquoi ClawdBot > Tout le Reste
- **Mémoire persistante** : Se souvient de tout, améliore ses propres fichiers de config
- **Proactivité** : Fait des choses sans qu'on le demande (basé sur les objectifs définis)
- **Universalité** : Code, écrit, research, browse, envoie des messages — tout dans un seul outil
- **Self-improvement** : Se reconfig lui-même, crée ses propres skills, améliore ses prompts

### Setup Phase 1 — Infrastructure
1. Choisir hardware (local > VPS)
2. `curl` install depuis openclaw.ai
3. Choisir modèle (Sonnet recommandé pour démarrer)
4. Configurer Telegram (BotFather → token → pair)

### Setup Phase 2 — Personalisation
1. Brain dump dans MEMORY.md
2. Remplir USER.md via interview : *"Interview me to fill in USER.md"*
3. Configurer SOUL.md avec les vraies directives business
4. Activer memory flush + session memory

### Setup Phase 3 — Optimisation
1. Configurer les muscles (Brave API, Codex, etc.)
2. Morning brief personnalisé
3. Discord ou Telegram channels par sujet
4. Premier cron job utile (daily brief ou monitoring)

### Setup Phase 4 — Automatisation
1. Mission Control (Next.js) buildé par le bot
2. Skills personnalisées pour les workflows récurrents
3. Subagents pour les tâches parallèles
4. Self-improvement loop dans le heartbeat

### Erreurs à Ne Jamais Faire
- Ne pas exposer à internet sans sécurité
- Ne pas éditer les config files à la main
- Ne pas lancer des workflows à fort impact sans test préalable
- Ne pas ignorer les logs quand quelque chose cloche

## 💡 Insights Actionnables
1. **Checklist setup** : Faire les 4 phases dans l'ordre, ne pas sauter à la phase 4 directement
2. **First week** : Brain dump + Morning brief uniquement. Ajouter la complexité progressivement.
3. **Learning loop** : Chaque semaine, demander *"Qu'est-ce qu'on peut améliorer dans notre workflow ?"*

## 🏪 Applications pour drinknellio.com
