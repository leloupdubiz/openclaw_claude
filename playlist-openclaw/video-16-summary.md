- **Avantage vitesse** : Inférence locale < 500ms vs 2-3s cloud → meilleur UX pour les workflows interactifs
- **Conseil** : Ne pas faire ça maintenant — commencer avec Mac Mini + Ollama, upgrader si le besoin est prouvé
